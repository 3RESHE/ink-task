Color

import FormatColor from '@stackpress/ink-ui/format/color';

Props

Configure the color display with these props. Use Ink utilities via `class` for additional styling.

Property Type Default Description value String - Color value (e.g., "red", "#FF0000") for the swatch and text label. box Boolean | Object false Shows a color swatch. If an object, accepts size props (e.g., `box-sm`, `box-md`). Defaults to 16px if true. text Boolean false Shows the color value as text. Automatically enables `box` if set without `box`. size String - Text size (e.g., "sm", "md", "lg") applied to the component. bold Boolean false Applies bold font weight to the text. display String "inline-flex" Display style ("flex", "inline-flex", "block", etc.). Aligns box and text if flex. class String - Ink utility classes for styling (e.g., "p-5 gap-10").

Basic Color

A color swatch with its value as text.
<format-color box-md text-md value="red" />

Box Only

A standalone color swatch without text.
<format-color box value="#00FF00" />

Text Only

The color value as text without a swatch.
<format-color text value="blue" box={false} />

Sized Color

A color display with custom box and text sizes.
<format-color box-lg text-xl value="purple" />

Bold Color

A color with bold text and a swatch.
<format-color box-sm text-md bold value="#FFA500" />
