Color

import Color from '@stackpress/ink-ui/format/color';

Props

The `<format-color>` component displays a color value as a swatch, text, or both. Use `box-sm`, `box-md`, or `box-lg` to change the size of the color swatch, and `text-sm`, `text-md`, or `text-lg` to adjust the text size.

Property Type Required Description value String Yes The color value to display (e.g., "red", "#FF0000"). Used as the background color of the swatch and text content if `text` is enabled. box Object/Boolean No If true, displays a color swatch. If an object or modifier (e.g., `box-md`), sets the swatch size. Defaults to false unless `text` is enabled. text Boolean No If true, displays the color value as text. If a modifier (e.g., `text-md`), sets the text size. Defaults to true if `box` is enabled. class String No Ink utility classes for styling (e.g., "p-4"). style String No Inline styles (use `class` with Ink utilities instead).

Swatch and Text

Displays both a color swatch and the color value as text in default and medium sizes.

<format-color value="red" class="p-4" /><br /> <format-color box-md text-md value="red" class="p-4" />

Swatch Only

Displays only the color swatch, without text, in default and large sizes.

<format-color box value="blue" text={false} class="p-4" /><br /> <format-color box-lg value="blue" text={false} class="p-4" />

Text Only

Displays only the color value as text, without a swatch, in default and small sizes.

<format-color value="green" box={false} text class="p-4" /><br /> <format-color value="green" box={false} text-sm class="p-4" />
