Currency

import Currency from '@stackpress/ink-ui/format/currency';

Props

The `<format-currency>` component displays a country flag and currency name based on a currency code. Use Ink utilities via the `class` prop for responsive styling. Note: Flag sizes default to 16px and text sizes map to font-size based on Ink's `xsizes` (e.g., xs: 10px, md: 14px, xl: 22px).

Property Type Required Description value String Yes The currency code (e.g., "usd", "php") to display. Special case: "usd" maps to "US" country code. flag Object | Boolean No If true, displays the country flag. If an object, specifies sizing (e.g., `{ size: "md" }`). Defaults to false. text Boolean No If true, displays the currency name. Defaults to true. flag-* String No Size modifier for the flag (e.g., "xs", "md", "xl"). Sets height (default: 16px if `flag` is true). text-* String No Size modifier for the text (e.g., "sm", "lg", "2xl"). Sets font-size (e.g., sm: 12px, lg: 18px). bold Boolean No If true, applies bold font weight to the text. Defaults to false. color String No Sets the text color (e.g., "var(--primary)", "#FF0000"). Defaults to inherited. display String No Sets display style (e.g., "inline-flex", "block"). Defaults to "inline-flex". class String No Ink utility classes for responsive styling (e.g., "p-4", "md:mr-10"). style String No Inline styles (use `class` with Ink utilities instead).

Basic Currency

A basic currency display with default flag and text sizes.
<format-currency value="usd" class="p-4" />

Flag-Only Currency

A currency display with only the flag, no name.
<format-currency flag value="php" text={false} class="p-4" />

Text-Only Currency

A currency display with only the name, no flag.
<format-currency value="eur" flag={false} text class="p-4" />

Small Currency Display

A currency display with small flag and text sizes.
<format-currency flag-xs text-xs value="jpy" class="p-4 b-solid b-t-1" />

Medium Currency Display

A currency display with medium flag and text sizes, with a border.
<format-currency flag-md text-md value="gbp" class="p-4 b-solid b-t-2 c-4" />

Large Currency Display

A currency display with large flag and text sizes.
<format-currency flag-lg text-lg value="cad" class="p-4 mr-10" />

Extra Large Currency Display

A currency display with extra-large flag and text sizes, bolded.
<format-currency flag-xl text-xl bold value="aud" class="p-4 b-solid b-t-3 c-6 shadow-2-4-6-8-000000" />

Colored Currency Display

A currency display with a custom text color and block display.
<format-currency flag-md text-md value="chf" color="var(--primary)" display="block" class="p-4 tx-center bg-t-1 rounded" />
