Country

import Country from '@stackpress/ink-ui/format/country';

Props

The `<format-country>` component displays a country flag and/or name based on a country code. Use Ink utilities via the `class` prop for responsive styling.

Property Type Required Description value String Yes The country code (e.g., "us", "ph") to display. Must match a valid code in `intl.json`. flag Object | Boolean No If true, displays the country flag. If an object, specifies sizing (e.g., `{ size: "md" }`). Defaults to false. text Boolean No If true, displays the country name. Defaults to true. flag-* String No Size modifier for the flag (e.g., "sm", "md", "lg"). Sets height (default: 16px if `flag` is true). text-* String No Size modifier for the text (e.g., "sm", "md", "lg"). Sets font-size (default: inherited if `text` is true). bold Boolean No If true, applies bold font weight to the text. Defaults to false. color String No Sets the text color (e.g., "red", "#FF0000", "var(--primary)"). Defaults to inherited. display String No Sets display style (e.g., "inline-flex", "block"). Defaults to "inline-flex". class String No Ink utility classes for responsive styling (e.g., "mr-10 p-4"). style String No Inline styles (use `class` with Ink utilities instead).

Basic Country

A basic country display with default flag and text sizes.
<format-country value="us" class="p-4" />

Flag-Only Country

A country display with only the flag, no name.
<format-country flag value="jp" text={false} class="p-4" />

Text-Only Country

A country display with only the name, no flag.
<format-country value="fr" flag={false} text class="p-4" />

Small Country Display

A country display with small flag and text sizes.
<format-country flag-xs text-xs value="it" class="p-4 b-solid b-t-1" />

Medium Country Display

A country display with medium flag and text sizes, with a border.
<format-country flag-md text-md value="es" class="p-4 b-solid b-t-2 c-4" />

Large Country Display

A country display with large flag and text sizes.
<format-country flag-lg text-lg value="de" class="p-4 mr-10" />

Extra Large Country Display

A country display with extra-large flag and text sizes, bolded.
<format-country flag-xl text-xl bold value="br" class="p-4 b-solid b-t-3 c-6 shadow-2-4-6-8-000000" />

Colored Country Display

A country display with a custom text color and block display.
<format-country flag-md text-md value="ph" color="var(--primary)" display="block" class="p-4 tx-center bg-t-1 rounded" />
