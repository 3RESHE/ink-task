Country

import CountryFormat from '@stackpress/ink-ui/format/country';

Props

Property Type Required Description value String Yes Two-letter country code (e.g., "us", "fr"). Case-insensitive. flag Boolean | Object No Displays the flag. If `true`, uses default size (16px). If an object, accepts size props (e.g., `sm`, `md`). Set to `false` to hide. text Boolean No Displays the country name. Defaults to `true` if `flag` is unset; set to `false` to hide. class String No Ink utility classes for styling the container (e.g., "flex gap-10"). display String No Display style (e.g., "inline-flex", "block"). Defaults to "inline-flex". color String No Text color using Ink utilities (e.g., "tx-primary"). size String No Font size using Ink utilities (e.g., "tx-md", "tx-lg"). bold Boolean No Applies bold font weight if `true`.

Basic Usage

Displays a country flag and name with default settings.
<format-country value="us" />

Flag Only

Shows only the country flag, hiding the name.
<format-country value="fr" flag text={false} />

Text Only

Shows only the country name, hiding the flag.
<format-country value="jp" flag={false} text />

Custom Styling

Customizes the flag size, text size, and adds styling with Ink utilities.
<format-country value="ca" flag-lg text-lg class="gap-10 tx-primary bold" />
