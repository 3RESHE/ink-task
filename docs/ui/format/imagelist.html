Image List

import ImageList from '@stackpress/ink-ui/format/imagelist';

Props

The `<format-imagelist>` component transforms an array of image sources from the `value` prop into a series of formatted `<img>` elements. Each item in the `value` array is used as the `src` attribute for an `<img>` tag, while additional props are processed into attributes (e.g., `alt`, `width`) with a default `loading="lazy"` applied if not specified. The container’s display format defaults to `block`, overridable via the `display` prop. Use the `class` prop with Ink utilities (e.g., `flex`, `gap-10`, `p-4`) to format the container and arrange the images.

Property Type Required Description value Array Yes An array of image source URLs or data URIs to format as `src` attributes for each `<img>` tag. display String No Defines the container’s display format (e.g., "block", "flex"). Defaults to "block". class String No Ink utility classes to format the container (e.g., "flex", "gap-10", "p-4"). style String No Inline styles for the container (use `class` with Ink utilities instead). Other attributes - No HTML attributes (e.g., `alt`, `width`, `height`, `loading`) formatted and passed to each `<img>` tag, with `loading="lazy"` as default if omitted.

Basic Image List Format

Renders a list of images with a basic `block` format and default lazy loading, sized consistently with `<format-image>`.
<format-imagelist value={['https://images.wsj.net/im-580612/8SR', 'https://images.wsj.net/im-580613/8SR', 'https://images.wsj.net/im-580614/8SR']} width="70" class="p-4" />

Attributed Image List Format

Renders a list of images with additional attributes, formatting each image with alt text and a consistent width.
<format-imagelist value={['https://images.wsj.net/im-580612/8SR', 'https://images.wsj.net/im-580613/8SR', 'https://images.wsj.net/im-580614/8SR']} alt="Image List Item" width="70" class="p-4 w-auto" />

Styled Image List Format

Renders a list of images with a styled container format, including padding and a border, with consistent image size.
<format-imagelist value={['https://images.wsj.net/im-580612/8SR', 'https://images.wsj.net/im-580613/8SR', 'https://images.wsj.net/im-580614/8SR']} alt="Styled Image" width="70" class="p-4 b-solid b-t-2 c-4" />

Flexible Image List Format

Renders a list of images with a flexible layout, arranging them horizontally with spacing, and consistent size.
<format-imagelist value={['https://images.wsj.net/im-580612/8SR', 'https://images.wsj.net/im-580613/8SR', 'https://images.wsj.net/im-580614/8SR']} display="flex" width="70" class="gap-10 p-4 tx-center bg-t-2 rounded" />
