Overflow

import Overflow from '@stackpress/ink-ui/format/overflow';

Props

The `<format-overflow>` component truncates a string from the `value` prop based on a specified `length`, either by character count (default) or word count (if `words` is `true`). An ellipsis (`…`) can be appended with the `hellip` prop. The component’s display defaults to `inline-block`, overridable via display props. Use styling props (`color`, `size`, `bold`) or the `class` prop with Ink utilities (e.g., `p-4`, `bg-t-2`) to customize appearance.

Property Type Required Description value String Yes The string to truncate. length Number/String No The maximum length (characters or words) to display. Converted to a number; defaults to `undefined` if invalid. words Boolean No If `true`, truncates by word count instead of character count (defaults to `false`). hellip Boolean No If `true`, appends an ellipsis (`…`) when the string is truncated (defaults to `false`). display String No Defines the container’s display format (e.g., "block", "flex", "inline-block"). Defaults to "inline-block". color String No Sets the text color (e.g., "var(--tx-1)", "red"). size String No Sets the font size (e.g., "16px", "14px"). bold Boolean No Applies bold styling to the text if `true`. class String No Ink utility classes to style the container (e.g., "p-4", "bg-t-2"). style String No Inline styles for the container (use `class` with Ink utilities instead).

Basic Overflow Format

Renders a truncated string with character-based length, matching the reference example.
<format-overflow value="Lorem Ipsum" length={8} hellip />

Word-Based Overflow Format

Renders a truncated string by word count with an ellipsis.
<format-overflow value="Lorem Ipsum is simply dummy text of the printing and typesetting industry." length={3} words hellip />

Styled Overflow Format

Renders a truncated string with custom styling (bold, color, size).
<format-overflow value="Lorem Ipsum" length={8} hellip bold color="var(--tx-2)" size="16px" />

Full Text Overflow Format

Renders the full string when the length exceeds the string length.
<format-overflow value="Lorem Ipsum" length={20} />
