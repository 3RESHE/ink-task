Phone

import Phone from '@stackpress/ink-ui/format/phone';

Props

The `<format-phone>` component transforms a phone number from the `value` prop into a clickable `tel:` link. The link text defaults to the `value` unless a `label` is provided. The component’s display defaults to `inline-block` with a font size of `13px`, overridable via display and size props. Use styling props (`color`, `size`, `bold`, `underline`) or the `class` prop with Ink utilities (e.g., `p-4`, `bg-t-2`) to customize appearance. Additional HTML attributes can be passed for the `<a>` tag.

Property Type Required Description value String Yes The phone number to format as a `tel:` link. label String No Custom text to display instead of the phone number (defaults to `value`). display String No Defines the container’s display format (e.g., "block", "flex", "inline-block"). Defaults to "inline-block". size String No Sets the font size (e.g., "16px", "14px"). Defaults to "13px". bold Boolean No Applies bold styling to the text if `true`. color String No Sets the link color (e.g., "var(--tx-1)", "red"). underline Boolean No Applies underline styling to the link if `true`. class String No Ink utility classes to style the container (e.g., "p-4", "bg-t-2"). style String No Inline styles for the container (ignored; use `class` with Ink utilities instead). ...attributes Object No Additional HTML attributes (e.g., `target`, `rel`) for the `<a>` tag.

Basic Phone Format

Renders a phone number as a clickable `tel:` link, matching the reference example.
<format-phone value="+63 (917) 555-2424" />

Custom Label Phone Format

Renders a phone link with a custom label instead of the phone number.
<format-phone value="+63 (917) 555-2424" label="Call Us" />

Styled Phone Format

Renders a phone link with custom styling (bold, color, underline).
<format-phone value="+63 (917) 555-2424" bold color="var(--tx-2)" underline />

Additional Attributes Phone Format

Renders a phone link with additional HTML attributes (e.g., `target` and `rel`).
<format-phone value="+63 (917) 555-2424" target="_blank" rel="noopener noreferrer" />
