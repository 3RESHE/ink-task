Number

import Number from '@stackpress/ink-ui/format/number';

Props

The `<format-number>` component transforms a numeric value from the `value` prop into a formatted string, applying custom separators, decimal points, and decimal places. The component’s display defaults to `inline-block`, overridable via display props. Use styling props (`color`, `size`, `bold`) or the `class` prop with Ink utilities (e.g., `p-4`, `bg-t-2`) to customize appearance.

Property Type Required Description value Number/String Yes The numeric value (or string representation) to format. separator String No The character to use as a thousands separator (defaults to an empty string, no separator). decimal String No The character to use as the decimal point (defaults to "."). decimals Number No The number of decimal places to display (defaults to the input’s decimal length). absolute Boolean No If `true`, removes the sign (positive or negative) from the output (defaults to `false`). display String No Defines the container’s display format (e.g., "block", "flex", "inline-block"). Defaults to "inline-block". color String No Sets the text color (e.g., "var(--tx-1)", "red"). size String No Sets the font size (e.g., "16px", "14px"). bold Boolean No Applies bold styling to the text if `true`. class String No Ink utility classes to style the container (e.g., "p-4", "bg-t-2"). style String No Inline styles for the container (use `class` with Ink utilities instead).

Basic Number Format

Renders a negative number with default formatting and separators, matching the reference example structure with a negative value.
<format-number value="-12345.67" separator="," decimal="." decimals={2} />

Custom Separator Format

Renders a number with a custom separator (e.g., space) and decimal settings.
<format-number value="1234567.895" separator=" " decimal="," decimals={3} />

Styled Number Format

Renders a number with custom styling (bold, color, size) and separators.
<format-number value="12345.67" separator="," decimal="." decimals={2} bold color="var(--tx-2)" size="16px" />

Absolute Number Format

Renders a negative number as absolute (without sign) with separators.
<format-number value="-12345.67" separator="," decimal="." decimals={2} absolute />
