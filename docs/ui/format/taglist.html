Taglist

import Taglist from '@stackpress/ink-ui/format/taglist';

Props

The `<format-taglist>` component renders an array of strings from the `value` prop as a list of `<element-badge>` components, styled as tags. It supports various shape options (`curved`, `rounded`, `pill`) and color themes (`info`, `warning`, `success`, `error`, `muted`, `primary`, `secondary`, `color`). The default theme is `warning` if no color is specified. The component’s display defaults to `inline-block`, overridable via display props. Use styling props or the `class` prop with Ink utilities (e.g., `p-4`, `bg-t-2`) to customize appearance.

Property Type Required Description value Array No The array of strings to render as tags (defaults to an empty array `[]`). curved Boolean No If `true`, applies a curved shape to the badges. rounded Boolean No If `true`, applies a rounded shape to the badges. pill Boolean No If `true`, applies a pill shape to the badges. info Boolean No If `true`, applies an info color theme. warning Boolean No If `true`, applies a warning color theme (default if no color specified). success Boolean No If `true`, applies a success color theme. error Boolean No If `true`, applies an error color theme. muted Boolean No If `true`, applies a muted color theme. primary Boolean No If `true`, applies a primary color theme. secondary Boolean No If `true`, applies a secondary color theme. color String No A custom color value (e.g., "var(--tx-1)", "red") to override themes. outline Boolean No If `true`, applies an outline style to the badges. solid Boolean No If `true`, applies a solid style to the badges. transparent Boolean No If `true`, applies a transparent style to the badges. display String No Defines the container’s display format (e.g., "block", "flex", "inline-block"). Defaults to "inline-block". class String No Ink utility classes to style the container (e.g., "p-4", "bg-t-2"). style String No Inline styles for the container (use `class` with Ink utilities instead).

Basic Taglist Format

Renders a taglist with pill-shaped badges, matching the reference example.
<format-taglist value={['Foo', 'bar']} pill />

Custom Color Taglist Format

Renders a taglist with a custom color.
<format-taglist value={['Foo', 'bar', 'baz']} pill success />

Styled Taglist Format

Renders a taglist with multiple styling options.
<format-taglist value={['Foo', 'bar']} pill success outline />

Default Theme Taglist Format

Renders a taglist with the default `warning` theme.
<format-taglist value={['Foo', 'bar', 'baz']} pill />

Transparent Taglist Format

Renders a taglist with a transparent style and pill shape.
<format-taglist value={['Foo', 'bar', 'baz']} pill transparent error />

Advanced Styled Taglist Format

Renders a taglist with a mix of styles, colors, and shapes.
<format-taglist value={['Foo', 'bar', 'baz']} rounded success outline solid />
