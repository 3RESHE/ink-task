HTML

import HTML from '@stackpress/ink-ui/format/html';

Props

The `<format-html>` component renders sanitized HTML content from the `value` prop, preventing XSS attacks using DOMPurify. Use Ink utilities via the `class` prop for responsive styling, including `tx-*` classes for font sizes (e.g., `tx-xs` for 10px, `tx-md` for 14px). Default font-size is inherited.

Property Type Required Description value String Yes The HTML content to render (sanitized by DOMPurify). class String No Ink utility classes for responsive styling (e.g., "p-4", "tx-md", "md:mr-10"). style String No Inline styles (use `class` with Ink utilities instead). Other attributes - No Additional HTML attributes are passed to the underlying `<span>` element.

Basic HTML

Renders a simple HTML string with default styling.
<format-html value="<p>Hello World</p>" class="p-4" />

Styled HTML

Renders HTML with inline styles that are preserved after sanitization.
<format-html value="<h1><strong style='color: green'>Hello</strong> World</h1>" class="p-4 b-solid b-t-2 c-4" />

Small HTML Display

Renders HTML with a smaller font size using `tx-xs` (10px).
<format-html value="<p>Small Text</p>" class="p-4 tx-xs b-solid b-t-1" />
