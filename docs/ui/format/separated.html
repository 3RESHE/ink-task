Separated

import Separated from '@stackpress/ink-ui/format/separated';

Props

The `<format-separated>` component renders an array from the `value` prop, separating items with a specified `separator`. If `separator` is `"line"`, items are displayed on new lines using `<div>` elements; otherwise, they are joined with the `separator` string. The component’s display defaults to `inline-block`, overridable via display props. Use styling props (`color`, `size`, `bold`) or the `class` prop with Ink utilities (e.g., `p-4`, `bg-t-2`) to customize appearance.

Property Type Required Description value Array No The array of items to separate (defaults to an empty array `[]`). separator String No The separator to use ("line" for new lines, or any string like "," or " "). Defaults to " ". display String No Defines the container’s display format (e.g., "block", "flex", "inline-block"). Defaults to "inline-block". color String No Sets the text color (e.g., "var(--tx-1)", "red"). size String No Sets the font size (e.g., "16px", "14px"). bold Boolean No Applies bold styling to the text if `true`. class String No Ink utility classes to style the container (e.g., "p-4", "bg-t-2"). style String No Inline styles for the container (use `class` with Ink utilities instead).

Basic Separated Format

Renders an array with items separated by new lines, matching the reference example.
<format-separated value={['Foo', 'bar']} separator="line" />

Custom Separator Format

Renders an array with a custom separator (e.g., comma).
<format-separated value={['Foo', 'bar', 'baz']} separator="," />

Styled Separated Format

Renders an array with custom styling (bold, color, size).
<format-separated value={['Foo', 'bar']} separator="line" bold color="var(--tx-2)" size="16px" />

Default Separator Format

Renders an array with the default space separator.
<format-separated value={['Foo', 'bar', 'baz']} />
