Metadata

import Metadata from '@stackpress/ink-ui/format/metadata';

Props

The `<format-metadata>` component transforms an object of key-value pairs from the `value` prop into a formatted table, displaying metadata in a two-column layout (key and value). The table spans the full width by default. Use props like `padding`, `align`, `background-theme`, `stripe-theme`, and `border-theme` to customize the appearance, and `format` to transform keys into a readable format (e.g., `created_at` to `Created At`). Additional styling can be applied via the `class` prop with Ink utilities.

Property Type Required Description value Object No An object of key-value pairs to display as metadata (defaults to an empty object). padding Number No Sets the padding of table cells in pixels (e.g., `10` for 10px). align String No Sets the text alignment of table cells (e.g., "center", "left", "right"). format Boolean No If `true`, formats keys by converting underscores/hyphens to spaces and capitalizing words (e.g., `created_at` to `Created At`). background-theme String No Sets the background color for all table cells using a theme value (e.g., "bg-1" maps to `var(--bg-1)`). Internally maps to the `background` prop. stripe-theme String No Sets the background color for odd-numbered rows to create a striping effect (e.g., "bg-2" maps to `var(--bg-2)`). Internally maps to the `stripe` prop. border-theme String No Sets the border-top color for table cells and enables a 1px solid border-top (e.g., "black"). Internally maps to the `border` prop. class String No Ink utility classes to style the container (e.g., "p-4", "bg-t-2"). style String No Inline styles for the container (use `class` with Ink utilities instead).

Basic Metadata Format

Renders a simple metadata table with default styling.
<format-metadata value={{ created_at: '2023-01-01', updated_at: '2023-12-31', author_name: 'John Doe' }} padding="10" align="center" />

Formatted Metadata Format

Renders a metadata table with formatted keys (e.g., `created_at` to `Created At`).
<format-metadata value={{ created_at: '2023-01-01', updated_at: '2023-12-31', author_name: 'John Doe' }} padding="10" align="center" format />

Styled Metadata Format

Renders a metadata table with custom padding, alignment, background, and striping.
<format-metadata value={{ created_at: '2023-01-01', updated_at: '2023-12-31', author_name: 'John Doe' }} padding="10" align="center" background-theme="bg-1" stripe-theme="bg-2" format />

Bordered Metadata Format

Renders a metadata table with a border-top and additional styling.
<format-metadata value={{ created_at: '2023-01-01', updated_at: '2023-12-31', author_name: 'John Doe' }} padding="10" align="center" background-theme="bg-1" stripe-theme="bg-2" border-theme="black" format />
