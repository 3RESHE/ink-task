Image

import Image from '@stackpress/ink-ui/format/image';

Props

The `<format-image>` component transforms an image source from the `value` prop into a formatted `<img>` element. The input `value` is used as the `src` attribute, while additional props are processed into attributes (e.g., `alt`, `width`) with a default `loading="lazy"` applied if not specified. The container’s display format defaults to `inline-block`, overridable via the `display` prop. Use the `class` prop with Ink utilities (e.g., `p-4`, `w-50`, `md:w-full`) to further format the container responsively.

Property Type Required Description value String Yes The image source URL or data URI to format as the `src` attribute. display String No Defines the container’s display format (e.g., "inline-block", "block"). Defaults to "inline-block". class String No Ink utility classes to format the container (e.g., "p-4", "w-50", "md:w-full"). style String No Inline styles for the container (use `class` with Ink utilities instead). Other attributes - No HTML attributes (e.g., `alt`, `width`, `height`, `loading`) formatted and passed to the `<img>` tag, with `loading="lazy"` as default if omitted.

Basic Image Format

Renders an image with a basic `inline-block` format and default lazy loading.
<format-image width="70" value="https://images.wsj.net/im-580612/8SR" class="p-4" />

Attributed Image Format

Renders an image with additional attributes, formatting it with alt text and a specified width.
<format-image value="https://images.wsj.net/im-580612/8SR" alt="Placeholder Image" width="200" class="p-4 w-auto" />

Styled Image Format

Renders an image with a styled container format, including padding and a border.
<format-image width="70" value="https://images.wsj.net/im-580612/8SR" alt="Styled Image" class="p-4 b-solid b-t-2 c-4" />

Block Image Format

Renders an image with a block-level format and centered alignment.
<format-image width="70" value="https://images.wsj.net/im-580612/8SR" display="block" class="p-4 tx-center bg-t-2 rounded" />
