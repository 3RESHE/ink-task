Table

import Table from '@stackpress/ink-ui/format/table';

Props

The `<format-table>` component renders an array of objects as a styled HTML table. Headers are derived from the first object’s keys, and rows from subsequent objects’ values. The component supports sticky positioning, custom padding, alignment, and theming for background, headers, stripes, and borders. The display defaults to `block`, overridable via display props. Use styling props or Ink utility classes (e.g., `p-4`, `bg-t-2`) to customize appearance.

Property Type Required Description value Array No An array of objects representing table rows (defaults to an empty array `[]`). top Boolean No If `true`, makes the header row sticky at the top. left Boolean No If `true`, makes the first column sticky on the left. right Boolean No If `true`, makes the last column sticky on the right. padding Number No Padding for cells in pixels (e.g., `10` for 10px). align String No Text alignment for cells ("left", "center", "right"). background String No Background color or theme for all cells (e.g., "bg-1"). border String No Border-top color or theme for cells (e.g., "black"). header String No Background color or theme for header cells (e.g., "bg-2"). stripe String No Background color or theme for alternating rows (e.g., "bg-2"). display String No Defines the container’s display format (e.g., "block", "flex"). Defaults to "block". color String No Sets the text color (e.g., "var(--tx-1)", "red"). class String No Ink utility classes to style the container (e.g., "p-4", "bg-t-2"). style String No Inline styles for the container (use `class` with Ink utilities instead).

Basic Table Format

Renders a table with custom padding, alignment, and theming, matching the reference example.
<format-table value={table} padding="10" align="center" background-theme="bg-1" stripe-theme="bg-2" header-theme="bg-2" border-theme="black" />

Sticky Header Table Format

Renders a table with a sticky header.
<format-table value={table} top padding="10" align="center" background-theme="bg-1" stripe-theme="bg-2" header-theme="bg-2" border-theme="black" />

Themed Table Format

Renders a table with custom colors and alignment.
<format-table value={table} padding="5" align="left" background="var(--bg-3)" stripe="var(--bg-4)" header="var(--bg-4)" border="var(--tx-1)" />

Simple Table Format

Renders a basic table with default styles.
<format-table value={table} />
