Email

import Email from '@stackpress/ink-ui/format/email';

Props

The `<format-email>` component creates a clickable email link (`mailto:`) with customizable styling and additional HTML attributes. The default font-size is 13px, overridable via the `size` prop or `tx-*` classes. The link is underlined by default, controllable via the `underline` prop.

Property Type Required Description value String Yes The email address for the `mailto:` link (e.g., "user@example.com"). label String No The display text for the link. Defaults to the `value` if not provided. bold Boolean No If true, applies bold font weight to the text. Defaults to false. color String No Sets the text color of the link (e.g., "var(--primary)", "#FF0000"). Defaults to inherited. underline Boolean No If true, underlines the link. Defaults to true. display String No Sets display style (e.g., "inline-block", "block"). Defaults to "inline-block". size String/Number No Sets the font size of the link (e.g., "12px", "1rem"). Defaults to 13px. class String No Ink utility classes for responsive styling (e.g., "p-4", "tx-md", "md:mr-10"). style String No Inline styles (use `class` with Ink utilities instead). Other attributes - No Additional HTML attributes (e.g., `target`, `rel`, `title`) are passed to the `<a>` tag.

Basic Email

A basic email link with the default 13px font size and underlined text.
<format-email value="user@example.com" class="p-4" />

Labeled Email

An email link with a custom label instead of the email address.
<format-email value="support@example.com" label="Contact Support" class="p-4" />

Small Email Display

An email link with a smaller font size using `tx-xs` (10px).
<format-email value="small@example.com" class="p-4 tx-xs b-solid b-t-1" />

Medium Email Display

An email link with a medium font size using `tx-md` (14px) and a border.
<format-email value="medium@example.com" class="p-4 tx-md b-solid b-t-2 c-4" />

Large Email Display

An email link with a larger font size using `tx-lg` (18px) and bold styling.
<format-email value="large@example.com" bold class="p-4 tx-lg mr-10" />

Colored Email Display

An email link with a custom text color using the `color` prop.
<format-email value="color@example.com" color="var(--primary)" class="p-4 tx-center bg-t-1 rounded" />

No Underline Email

An email link with the underline removed using `underline={false}`.
<format-email value="nounderline@example.com" underline={false} class="p-4 b-solid b-t-2 c-4" />

Block Email Display

An email link as a block element with centered text and a background.
<format-email value="block@example.com" display="block" class="p-4 tx-center bg-t-2 rounded" />

Responsive Email Display

An email link that adjusts font size based on screen size (small on mobile, large on desktop).
<format-email value="responsive@example.com" class="p-4 md:tx-sm lg:tx-lg" />

Enhanced Email with Attributes

An email link with additional HTML attributes like `target` and `title` for enhanced functionality.
<format-email value="enhanced@example.com" label="Email Support" target="_blank" title="Opens email client in a new tab" class="p-4 b-solid b-t-2 c-4" />
