Email

import EmailFormat from '@stackpress/ink-ui/format/email';

Props

Customize the email link with the props below. Additional HTML attributes are passed to the `<a>` tag.

Property Type Required Description value String Yes Email address (e.g., "john@doe.com"). Used in the `mailto:` link and as the default display text. label String No Custom text to display instead of the email address. class String No Ink utility classes for styling the container (e.g., "tx-md"). display String No Display style for the container (e.g., "inline-block", "block"). Defaults to "inline-block". size String No Font size using Ink utilities (e.g., "md", "lg"). Defaults to "13px". bold Boolean No Applies bold font weight to the link if `true`. color String No Text color for the link using Ink utilities (e.g., "primary", "secondary"). underline Boolean No Underlines the link if `true`. [html-attributes] Any No Additional HTML attributes (e.g., `target`, `rel`) passed to the `<a>` tag.

Basic Usage

Displays an email address as a clickable mailto link with default styling.
<format-email value="john@doe.com" />

Custom Label

Displays a custom label instead of the email address.
<format-email value="jane@doe.com" label="Contact Jane" />

Styled Email

Styles the email link with size, color, and underline options.
<format-email primary underline md value="john@doe.com" />

Link Attributes

Adds additional HTML attributes to the link, like `target` or `rel`.
<format-email value="support@doe.com" target="_blank" rel="noopener" bold />
