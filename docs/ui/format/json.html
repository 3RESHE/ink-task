JSON

import JSONFormat from '@stackpress/ink-ui/format/json';

Props

The `<format-json>` component transforms a JSON-serializable value from the `value` prop into a formatted JSON string, displayed within a `<pre>` tag for readability. The container’s display format defaults to `block`, overridable via display props (e.g., `block`, `flex`, `inline-block`). Use the `class` prop with Ink utilities (e.g., `p-4`, `bg-t-2`, `scroll-y-auto`) to style the container and manage overflow or text formatting.

Property Type Required Description value Any Yes A JSON-serializable value (e.g., object, array, string) to format and display as a JSON string. display String No Defines the container’s display format (e.g., "block", "flex", "inline-block"). Defaults to "block". Can be set via props like `block`, `flex`, `inline-block`, etc. class String No Ink utility classes to style the container (e.g., "p-4", "bg-t-2", "scroll-y-auto"). style String No Inline styles for the container (use `class` with Ink utilities instead).

Basic JSON Format

Renders a simple JSON object with a basic `block` format.
<format-json value={{ icon: 'book', label: 'Docs' }} class="p-4" />

Array JSON Format

Renders a JSON array, demonstrating handling of different data types.
<format-json value={['apple', 'banana', 'cherry']} class="p-4" />

Styled JSON Format

Renders a JSON object with a styled container, including padding, background, and borders.
<format-json value={{ icon: 'book', label: 'Docs' }} class="p-4 bg-t-2 b-solid b-t-2 c-4" />

Scrollable JSON Format

Renders a larger JSON object with a scrollable container and custom display.
<format-json value={{ id: 1, name: 'Example', items: [{ id: 101, name: 'Item 1', active: true }, { id: 102, name: 'Item 2', active: false }, { id: 103, name: 'Item 3', active: true }], metadata: { created: '2023-01-01', updated: '2023-12-31' } }} flex class="p-4 tx-14 scroll-y-auto h-100 w-full bg-t-2 rounded" />
