List

import List from '@stackpress/ink-ui/format/list';

Props

The `<format-list>` component transforms an array of strings from the `value` prop into a formatted list, rendered as either an `<ol>` (ordered) or `<ul>` (unordered) based on the `ordered` prop. The container’s display defaults to `block`, overridable via display props. Use `indent` and `spacing` props to customize the list’s padding, or the `class` prop with Ink utilities (e.g., `p-4`, `bg-t-2`) to style the container.

Property Type Required Description value Array Yes An array of strings to render as list items. ordered Boolean No If `true`, renders an ordered list (`<ol>`); if `false` or omitted, renders an unordered list (`<ul>`). Defaults to `false`. indent Number No Sets the left padding of the list in pixels (e.g., `20` for 20px). spacing Number No Sets the vertical padding between list items in pixels (e.g., `10` for 10px). display String No Defines the container’s display format (e.g., "block", "flex"). Defaults to "block". class String No Ink utility classes to style the container (e.g., "p-4", "bg-t-2"). style String No Inline styles for the container (use `class` with Ink utilities instead).

Basic Unordered List Format

Renders a simple unordered list (`<ul>`) with default `block` display.
<format-list value={['Item 1', 'Item 2', 'Item 3']} class="p-4" />

Ordered List Format

Renders an ordered list (`<ol>`) with numbered items.
<format-list value={['Item 1', 'Item 2', 'Item 3']} ordered class="p-4" />

Indented List Format

Renders an unordered list with custom indentation.
<format-list value={['Item 1', 'Item 2', 'Item 3']} indent={20} class="p-4" />

Spaced List Format

Renders an ordered list with custom spacing between items.
<format-list value={['Item 1', 'Item 2', 'Item 3']} ordered spacing={10} class="p-4" />
