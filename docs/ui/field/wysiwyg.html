WYSIWYG

import WYSIWYG from '@stackpress/ink-ui/field/wysiwyg';

Props

The `<field-wysiwyg>` component provides a rich text editor for creating formatted content. Use Ink utilities via the `class` prop to style it responsively. Below are its props:

Property Type Required Description name String No Name attribute for form submission. value String No Initial content for the editor (defaults to ""). disabled Boolean No Disables the editor, preventing interaction. readonly Boolean No Makes the editor read-only, allowing view only. history Boolean No Enables undo/redo buttons (default: false). font Boolean No Enables font family selection (default: false). size Boolean No Enables font size selection (default: false). format Boolean No Enables format block selection (e.g., headings, paragraph; default: false). paragraph Boolean No Enables paragraph style options (default: false). blockquote Boolean No Enables blockquote button (default: false). style Boolean No Enables text style buttons (bold, italic, etc.; default: false). color Boolean No Enables font color selection (default: false). highlight Boolean No Enables highlight color selection (default: false). text Boolean No Enables text style options (default: false). remove Boolean No Enables remove format button (default: false). indent Boolean No Enables indent/outdent buttons (default: false). align Boolean No Enables alignment options (default: false). rule Boolean No Enables horizontal rule button (default: false). list Boolean No Enables ordered/unordered list buttons (default: false). lineheight Boolean No Enables line height selection (default: false). table Boolean No Enables table creation button (default: false). link Boolean No Enables link insertion button (default: false). image Boolean No Enables image insertion button (default: false). video Boolean No Enables video insertion button (default: false). audio Boolean No Enables audio insertion button (default: false). fullscreen Boolean No Enables fullscreen toggle button (default: false). showblocks Boolean No Enables show blocks toggle button (default: false). code Boolean No Enables code view toggle button (default: false). print Boolean No Enables preview and print buttons (default: false). save Boolean No Enables save and template buttons (default: false). dir Boolean No Enables direction toggle buttons (LTR/RTL; default: false). change Function No Custom handler for change events, receiving the event. update Function No Callback with the updated content. class String No Ink utility classes for responsive styling (e.g., "w-full p-5 b-solid b-t-3 c-6"). style String No Inline styles (use `class` with Ink utilities instead).

Basic WYSIWYG Editor

A basic WYSIWYG editor with default Ink styling and common toolbar options.
<field-wysiwyg name="content" history font size format style color align list link class="w-full p-4 b-solid b-t-2 c-4 tx-md" style="cursor: text;" change={(e) => console.log('Change event:', e.target.value)} update={(value) => console.log('Updated value:', value)} />

Styled WYSIWYG Editor

A styled WYSIWYG editor with responsive Ink utilities and hover effects.
<field-wysiwyg name="article" history font size format style color align list link class="w-full min-h-300 p-6 b-solid b-t-3 c-6 tx-lg md:vw-90 sm:vw-95 shadow-md bg-white transition-300 hover:b-primary hover:shadow-0-2-8-t-3" style="cursor: text;" change={handleChange} update={handleUpdate} />

Minimal WYSIWYG Editor

A minimal WYSIWYG editor with limited toolbar options and Ink styling.
<field-wysiwyg name="note" style color align class="w-full min-h-200 p-4 b-dashed b-t-1 c-4 tx-verdana" style="cursor: text;" update={(value) => console.log('Updated value:', value)} />

Full-Featured WYSIWYG Editor

A full-featured WYSIWYG editor with all toolbar options, styled with Ink utilities and hover effects.
<field-wysiwyg name="document" history font size format paragraph blockquote style color highlight text remove indent align rule list lineheight table link image video audio fullscreen showblocks code print save dir class="w-full min-h-400 p-8 b-solid b-t-4 c-8 tx-xl md:vw-90 sm:vw-95 lg:shadow-lg transition-300 hover:b-primary" style="cursor: text;" change={(e) => console.log('Change event:', e.target.value)} update={handleUpdate} />

Pre-filled WYSIWYG Editor

A WYSIWYG editor pre-filled with initial content, styled with Ink.
<field-wysiwyg name="blog" value="<h1>Welcome</h1><p>This is a <b>pre-filled</b> editor with some content.</p>" history font size style color align list link class="w-full min-h-300 p-6 b-solid b-t-3 c-6 tx-md transition-300 hover:b-primary" style="cursor: text;" change={handleChange} update={(value) => console.log('Updated value:', value)} />

Disabled WYSIWYG Editor

A disabled WYSIWYG editor with pre-filled content and muted styling.
<field-wysiwyg name="lockedContent" value="<p>This content is <i>locked</i> and cannot be edited.</p>" disabled class="w-full min-h-200 p-4 b-solid b-muted c-5 tx-muted" style="cursor: not-allowed;" />
