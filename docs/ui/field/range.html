Range

import Range from '@stackpress/ink-ui/field/range';

Props

The `<field-range>` component provides a range slider input for selecting a numeric value within a defined range. Below are its props:

Property Type Required Description name String No Name attribute for form submission. value String/Number No Initial value of the range input. min String/Number No Minimum value (defaults to 0). max String/Number No Maximum value (defaults to 100). step String/Number No Step interval (defaults to 1). width String/Number No Width of the range slider in pixels (defaults to browser default). change Function No Custom handler for change events, receiving the selected value. update Function No Callback with the updated numeric value. autocomplete String No Controls browser autofill behavior. disabled Boolean No Disables the input. readonly Boolean No Makes the input read-only. required Boolean No Marks the input as required. class String No CSS classes for the host element (use Ink utilities). style String No Inline styles (prefer Ink utilities instead).

Basic Range Input

A basic range input with default settings.
<field-range name="volume" value="50" />

Range with Limits

A range input with custom `min`, `max`, and `step` settings.
<field-range name="temperature" value="25" min="0" max="50" step="5" />

Styled Range Input

A styled range input with custom width, dynamic styling, and smooth transitions using Ink utilities and inline styles.
<field-range name="brightness" value="75" class="w-300 bg-transparent tx-t-1 h-8 c-5 transition-300" style="background: linear-gradient(to right, var(--primary) 75%, var(--t-2) 75%); accent-color: var(--primary); cursor: pointer;" onmouseover="this.style.boxShadow='0 2px 8px var(--t-3)'; this.style.opacity='0.9'" onmouseout="this.style.boxShadow='none'; this.style.opacity='1'" />
