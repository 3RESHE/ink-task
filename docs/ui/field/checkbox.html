Basic Usage

A simple checkbox with a label.
<field-checkbox name="basic" label=" Agree to terms" />

States

Checkbox with different states: checked, disabled, required.
<field-checkbox name="checked" label="Checked" value="yes" checked /> <field-checkbox name="disabled" label="Disabled" disabled /> <field-checkbox name="required" label="Required" required />

Shape

Checkbox with different shapes: default (square), rounded, circle.
<field-checkbox name="square" label="Checked" value="yes" checked /> <field-checkbox name="rounded" label="Rounded" value="yes" rounded circle/> <field-checkbox name="circle" label="Square" value="yes" square />

Custom Styling

Checkbox with custom color and update callback.
<field-checkbox name="custom" label="Active?" value="yes" checked orange update={(e) => console.log('Checkbox changed:', e.target.checked)} />
