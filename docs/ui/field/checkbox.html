Checkbox

import Checkbox from '@stackpress/ink-ui/field/checkbox';

Props

Property Type Required Notes label String No Text displayed next to the checkbox. Defaults to an empty string. checked Boolean No If true, the checkbox is initially checked. disabled Boolean No If true, disables the checkbox. name String No Name attribute for form submission. readonly Boolean No If true, makes the checkbox read-only. required Boolean No If true, marks the checkbox as required for form validation. value String No Value submitted when the checkbox is checked. click Function No Custom click event handler for the checkbox. display String No Sets display style of the host (e.g., "inline-block", "block"). Defaults to "inline-block". orange Boolean No If true, applies an orange color variant to the checkbox. rounded Boolean No If true, applies rounded styling to the checkbox. square Boolean No If true, applies square styling to the checkbox (assumed supported by setStyles). class String No Custom CSS classes for the host element (e.g., "mr-10"). style String No Inline styles for the host element (e.g., "margin: 10px").

Basic Checkbox

A simple checkbox with a label.
<field-checkbox name="active" label="Active" />

Checked Checkbox

A checkbox that is initially checked.
<field-checkbox name="agree" label="I Agree" checked />

Disabled Checkbox

A disabled checkbox that cannot be interacted with.
<field-checkbox name="terms" label="Accept Terms" disabled />

Styled Checkbox

A checkbox with custom styling, such as color and shape variants.
<field-checkbox name="subscribe" label="Subscribe" orange rounded />

Checkbox with Handler

A checkbox with a custom click event handler.
<script> const handleClick = (e) => console.log('Clicked:', e.target.checked); </script> <field-checkbox name="notify" label="Notify Me" click={handleClick} />

Required Checkbox

A checkbox marked as required for form submission.
<field-checkbox name="consent" label="Consent Required" required />

Checkbox with Value

A checkbox with a specific value logged on change.
<script> const logValue = (e) => console.log('Value:', e.target.checked ? e.target.value : 'unchecked'); </script> <field-checkbox name="feature" label="Enable Feature" value="feature_on" click={logValue} />

Shape and Color Combo

A checkbox combining different shapes and colors.
<field-checkbox name="option1" label="Rounded Orange" orange rounded /> <field-checkbox name="option2" label="Square Default" square />

Multiple Checkboxes in Form

A group of checkboxes in a form with varied states and styles.
<field-checkbox name="hobbies" label="Reading" value="reading" checked orange rounded /> <field-checkbox name="hobbies" label="Gaming" value="gaming" /> <field-checkbox name="hobbies" label="Traveling" value="traveling" disabled />
