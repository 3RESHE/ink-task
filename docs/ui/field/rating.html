Rating Field

import RatingField from '@stackpress/ink-ui/field/rating';

Props

The `<field-rating>` component allows users to set a rating using stars. Below are its supported props:

Name Type Required Notes name String No Name attribute for form submission value String | Number No Initial rating value (default: 0) max String | Number No Maximum number of stars (default: 5) class String No CSS classes for styling (e.g., size, color via Ink utilities) style String No Inline styles (prefer Ink utilities)

Basic Rating

A basic rating input with 5 stars.
<field-rating name="basic-rating" value="3" />

Custom Max Stars

Rating input with a custom maximum number of stars.
<field-rating name="custom-rating" value="4" max="7" />

Custom Size and Color

Rating input styled with larger size and custom color.
<field-rating name="styled-rating" value="2" max="5" class="tx-24 tx-primary hover:tx-t-2 transition-300" />
