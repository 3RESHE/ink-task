Textarea

import Textarea from '@stackpress/ink-ui/field/textarea';

Props

The `<field-textarea>` component provides a textarea input for multi-line text. Below are its props:

Property Type Required Description name String No Name attribute for form submission. value String No Initial value of the textarea (can also be set via inner content). placeholder String No Placeholder text for the textarea. disabled Boolean No Disables the textarea. readonly Boolean No Makes the textarea read-only. required Boolean No Marks the textarea as required. rows Number No Number of visible rows in the textarea (default: 2). change Function No Custom handler for change events, receiving the event. update Function No Callback with the updated value. class String No CSS classes for the host element (use Ink utilities). style String No Inline styles (prefer Ink utilities instead).

Basic Textarea Input

A basic textarea input with a placeholder and initial value.
<field-textarea name="description" value="Initial description text." placeholder="Enter your description..." rows={3} />

Readonly Textarea Input

A readonly textarea input with pre-filled content.
<field-textarea readonly rows={4} value="This is a readonly textarea. You cannot edit this content." />

Styled Textarea Input

A styled textarea input with custom styling, hover effects, and event handlers.
<field-textarea name="notes" placeholder="Add your notes..." rows={5} class="w-300 p-5 b-solid b-t-2 c-5 tx-lg tx-t-1 bg-white transition-300 hover:b-primary hover:shadow-0-2-8-t-3" style="cursor: text;" change={(e) => console.log('Change event:', e.target.value)} update={(value) => console.log('Updated value:', value)} />

Disabled Textarea Input

A disabled textarea input with pre-filled content and cursor styling.
This textarea is disabled. You cannot interact with it.
<field-textarea disabled rows={3} class="w-300 tx-muted cursor-not-allowed" > This textarea is disabled. You cannot interact with it. </field-textarea>

Required Textarea Input

A required textarea input with minimal rows and simple styling.
<field-textarea name="feedback" placeholder="Your feedback is required..." required rows={2} class="w-250 p-3 b-solid b-error c-4" update={(value) => console.log('Updated value:', value)} />
