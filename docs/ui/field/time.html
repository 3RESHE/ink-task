Time

import Time from '@stackpress/ink-ui/field/time';

Props

The `<field-time>` component provides a time input field for selecting hours and minutes. Below are its props:

Property Type Required Description autocomplete String No Controls autocomplete behavior (e.g., "on", "off"; default: "off"). disabled Boolean No Disables the time input. name String No Name attribute for form submission. pattern String No Regex pattern for input validation (e.g., "[0-2][0-9]:[0-5][0-9]"). readonly Boolean No Makes the time input read-only. required Boolean No Marks the time input as required. value String | Date No Initial value for the time input (e.g., "14:30" or Date object; default: empty). change Function No Custom handler for change events, receiving the event. update Function No Callback with the updated value. class String No CSS classes for the host element (use Ink utilities). style String No Inline styles (prefer Ink utilities instead).

Basic Time Input

A basic time input with no initial value and event handlers.
<field-time name="appointment" change={(e) => console.log('Change event:', e.target.value)} update={(value) => console.log('Updated value:', value)} />

Styled Time Input

A styled time input with custom styling, hover effects, and event handlers.
<field-time name="meeting" class="w-200 p-5 b-solid b-t-2 c-5 tx-md tx-t-1 bg-white transition-300 hover:b-primary hover:shadow-0-2-8-t-3" style="cursor: text;" change={(e) => console.log('Change event:', e.target.value)} update={(value) => console.log('Updated value:', value)} />

Disabled Time Input

A disabled time input with a pre-filled value and muted styling.
<field-time name="event" disabled value="18:00" class="w-200 p-5 tx-muted b-solid b-t-1 c-4" style="cursor: not-allowed;" />

Required Time Input

A required time input with error styling and autocomplete disabled.
<field-time name="deadline" required autocomplete="off" class="w-200 p-5 b-solid b-error c-4 tx-md tx-t-2" style="cursor: text;" update={(value) => console.log('Updated value:', value)} />

Pre-filled Time Input

A pre-filled time input with a Date object value and hover effects.
<field-time name="startTime" value={new Date('2023-01-01T09:30:00')} class="w-200 p-5 b-dashed b-t-1 c-6 tx-verdana tx-bold transition-300 hover:b-primary" style="cursor: text;" change={(e) => console.log('Change event:', e.target.value)} update={(value) => console.log('Updated value:', value)} />

Readonly Time Input

A readonly time input with a pattern restriction and muted styling.
<field-time name="fixedTime" readonly value="14:00" pattern="[0-2][0-9]:[0-5][0-9]" class="w-200 p-5 b-solid b-muted c-5 tx-muted tx-italic" style="cursor: default;" />
