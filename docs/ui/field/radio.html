Radio

import Radio from '@stackpress/ink-ui/field/radio';

Props

The `<field-radio>` component provides a radio button input for selecting a single option from a group. Below are its props:

Property Type Required Description name String No Name attribute linking radio buttons in a group for form submission. value String No Value of the radio button, submitted when selected. checked Boolean No Sets the initial checked state of the radio button. disabled Boolean No Disables the radio button, preventing interaction. required Boolean No Marks the radio group as required (enforced at form level). change Function No Custom handler for change events, receiving the selected value. class String No CSS classes for the host element (use Ink utilities). style String No Inline styles for the host element (prefer Ink utilities).

Basic Radio Group

A simple radio group with multiple options.
Yes No
<field-radio name="option" value="yes">Yes</field-radio> <field-radio name="option" value="no">No</field-radio>

Radio with Default Selection

A radio group with a pre-selected option using the `checked` prop.
Yes No
<field-radio name="choice" value="yes" checked>Yes</field-radio> <field-radio name="choice" value="no">No</field-radio>

Disabled Radio Button

A radio group with a disabled option.
Active Inactive
<field-radio name="status" value="active">Active</field-radio> <field-radio name="status" value="inactive" disabled>Inactive</field-radio>
