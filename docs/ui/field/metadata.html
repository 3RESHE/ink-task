Metadata

import Metadata from '@stackpress/ink-ui/field/metadata';

Props

The `<field-metadata>` component allows dynamic addition and removal of key-value pairs for form submission. Below are its props:

Property Type Required Description name String No Base name for form submission (e.g., `metadata[key]`). add String No Text for the "Add" button. Defaults to "Add". value Object No Initial key-value pairs to populate the component. Defaults to `{}`. class String No CSS classes for the host element, styled via default utilities. style String No Inline styles for the host element, overriding defaults.

Basic Metadata

A simple metadata field with default settings, allowing users to add key-value pairs.
<field-metadata name="metadata" />

Metadata with Initial Values

Pre-populates the metadata field with initial key-value pairs using the `value` prop.
<field-metadata name="metadata" value={{ "author": "John Doe", "date": "2023-10-01" }} />

Custom Add Button Text

Customizes the "Add" button text using the `add` prop.
<field-metadata name="metadata" add="Add New Pair" />

Styled Metadata

Applies custom styles to the metadata field using `class` and `style` props.
<field-metadata name="metadata" class="w-300 bg-white tx-t-1 c-10 p-15 b-solid b-t-2 transition-300 hover:b-primary" style="border-color: var(--t-2); background: linear-gradient(135deg, #ffffff, #f9f9f9); box-shadow: 0 2px 6px var(--t-1);" onmouseover="this.style.borderColor='var(--primary)'; this.style.boxShadow='0 4px 12px var(--t-3)'" onmouseout="this.style.borderColor='var(--t-2)'; this.style.boxShadow='0 2px 6px var(--t-1)'" />
