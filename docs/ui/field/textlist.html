Textlist

import Textlist from '@stackpress/ink-ui/field/textlist';

Props

The `<field-textlist>` component provides a dynamic list of text inputs that can be added or removed. Below are its props:

Property Type Required Description name String No Name attribute for form submission (applied to each input). add String No Label for the "Add" button (defaults to "Add"). value Array<String> No Initial list of text values for the textlist (defaults to []). class String No CSS classes for the host element (use Ink utilities). style String No Inline styles (prefer Ink utilities instead).

Basic Textlist Input

A basic textlist input with a default "Add" button and placeholder.
<field-textlist name="items" placeholder="Enter an item..." change={(e) => console.log('Change event:', e.target.value)} update={(value) => console.log('Updated value:', value)} />

Styled Textlist Input

A styled textlist input with custom border, padding, and hover effects.
<field-textlist name="tasks" add="Add Task" placeholder="Enter a task..." class="w-350 p-5 b-dashed b-t-2 c-6 tx-md tx-t-1 bg-white transition-300 hover:b-primary hover:shadow-0-2-8-t-3" style="cursor: text;" change={(e) => console.log('Change event:', e.target.value)} update={(value) => console.log('Updated value:', value)} />

Pre-filled Textlist Input

A textlist input pre-filled with initial values and subtle styling.
<field-textlist name="todos" value={["Buy groceries", "Finish project", "Call friend"]} placeholder="Add another todo..." class="w-300 b-solid b-t-1 c-4 transition-300 hover:b-primary" style="cursor: text;" update={(value) => console.log('Updated value:', value)} />

Compact Textlist Input

A compact textlist input with minimal styling and centered text inputs.
<field-textlist name="notes" add="+" value={["Note 1", "Note 2"]} placeholder="Add a note..." class="w-250 p-3 tx-center b-solid b-muted c-3" style="cursor: text;" update={(value) => console.log('Updated value:', value)} />
