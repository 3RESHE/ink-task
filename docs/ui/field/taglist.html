Taglist

import Taglist from '@stackpress/ink-ui/field/taglist';

Props

The `<field-taglist>` component provides a tag input field for managing tags. Below are its props:

Property Type Required Description name String No Name attribute for form submission (generates hidden inputs per tag). value Array/String No Initial tags as an array or comma-separated string (default: `[]`). placeholder String No Placeholder text for the input field. disabled Boolean No Disables the taglist input. readonly Boolean No Makes the taglist read-only. required Boolean No Marks the taglist as required. info Boolean No Sets info color for tags. warning Boolean No Sets warning color for tags (default true). success Boolean No Sets success color for tags. error Boolean No Sets error color for tags. muted Boolean No Sets muted color for tags. primary Boolean No Sets primary color for tags. secondary Boolean No Sets secondary color for tags. color String No Custom color for tags (e.g., "#ff4444"; overrides other color props). change Function No Custom handler for change events on the input, receiving the event. update Function No Callback with the updated tag list (array) after add/remove. class String No CSS classes for the host element (use Ink utilities). style String No Inline styles (prefer Ink utilities instead).

Basic Taglist Input

A basic taglist input with default warning color.
<field-taglist name="tags" value={["tag1", "tag2"]} warning />

Themed Taglist Input

A themed taglist input with success color and placeholder.
<field-taglist name="skills" placeholder="Add skills..." success value="coding,design" />

Styled Taglist Input

A styled taglist input with primary color, custom styling, and hover effects.
<field-taglist name="categories" primary placeholder="Enter categories..." value={["tech", "news"]} class="w-300 b-solid b-t-2 c-5 tx-lg bg-white transition-300 hover:b-primary hover:shadow-0-2-8-t-3" style="cursor: text;" change={(e) => console.log('Change event:', e.target.value)} update={(value) => console.log('Updated tags:', value)} />

Error-Themed Taglist with Custom Color

A taglist with error theme overridden by a custom color.
<field-taglist name="errors" placeholder="Add error tags..." error color="#ff4444" value={["bug", "crash"]} update={(value) => console.log('Updated tags:', value)} />

Muted Taglist with String Input

A taglist with muted color theme using a comma-separated string input.
<field-taglist name="labels" placeholder="Add labels..." muted value="urgent,pending,review" class="w-250" update={(value) => console.log('Updated tags:', value)} />

Secondary Taglist with Extensive Styling

A taglist with secondary color, extensive styling, and disabled state.
<field-taglist name="topics" secondary placeholder="Add topics..." value={["ai", "ml", "data"]} disabled class="w-300 p-5 b-solid b-t-2 c-5 tx-lg tx-muted bg-t-0 transition-300" style="cursor: not-allowed;" update={(value) => console.log('Updated tags:', value)} />
