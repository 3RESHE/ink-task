Date

import Date from '@stackpress/ink-ui/field/date';

Props

Property Type Required Notes autocomplete String No Sets autocomplete behavior (e.g., "off"). disabled Boolean No If true, disables the date input. name String No Name attribute for form submission. pattern String No Validation pattern, though less relevant for type="date". readonly Boolean No If true, makes the date input read-only. required Boolean No If true, marks the date input as required. value String | Date No Initial date value (e.g., "2025-03-07" or Date object). Normalized to "YYYY-MM-DD". change Function No Custom change event handler, receives the event. update Function No Callback with normalized date string (e.g., "2025-03-07"). class String No Custom CSS classes for the host element (e.g., "w-200"). style String No Inline styles for the host element (e.g., "border: 2px solid blue").

Basic Date (name)

A simple date picker with a form name.
<field-date name="date" />

Date with Value

A date picker with an initial value.
<field-date name="date" value="2025-03-07" />

Disabled Date

A disabled date picker with a value.
<field-date name="date" value="2025-03-07" disabled />

Readonly Date

A read-only date picker with a value.
<field-date name="date" value="2025-03-07" readonly />

Required Date

A date picker marked as required.
<field-date name="date" required />

Autocomplete Date

A date picker with autocomplete disabled.
<field-date name="date" autocomplete="off" />

Pattern Date

A date picker with a pattern (though less effective for type="date").
<field-date name="date" pattern="\d{4}-\d{2}-\d{2}" />

Date with Change

A date picker with a custom change handler.
<script> const handleChange = (e) => console.log('Change event:', e.target.value); </script> <field-date name="date" change={handleChange} />

Date with Update

A date picker with an update callback.
<script> const handleUpdate = (date) => console.log('Updated date:', date); </script> <field-date name="date" update={handleUpdate} />

Styled Date (class, style)

A styled date picker with custom class and inline styles.
<field-date name="date" value="2025-03-07" class="w-250 relative z-1 bg-white rounded-5 tx-t-1 p-10 tx-16 b-solid b-primary" style="border-width: 2px; transition: all 0.3s ease;" onmouseover="this.style.borderColor='var(--primary-dark)'; this.style.boxShadow='0 2px 8px var(--t-3)'" onmouseout="this.style.borderColor='var(--primary)'; this.style.boxShadow='none'" />
