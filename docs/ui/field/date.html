Date Field

import DateField from '@stackpress/ink-ui/field/date';

Props

Name Type Required Notes autocomplete String No Autocomplete attribute (e.g., "off") disabled Boolean No Disables the date picker name String No Name attribute for form submission pattern String No Regex pattern for validation (e.g., "[0-9]{4}-[0-9]{2}-[0-9]{2}") readonly Boolean No Makes the date picker read-only required Boolean No Marks the field as required value String | Date No Initial date value (e.g., "2020-01-01" or Date object), converted to YYYY-MM-DD placeholder String No Placeholder text for the date input change Function No Callback for change event (receives ChangeEvent) update Function No Callback with formatted date string (YYYY-MM-DD)

Basic Usage

A simple date field with a placeholder.
<field-date name="basic-date" placeholder="Select a date" />

States

Date field with different states: value, disabled, required.
<field-date name="valued-date" value="2023-06-15" placeholder="With Value" /> <field-date name="disabled-date" disabled placeholder="Disabled" /> <field-date name="required-date" required placeholder="Required" />

Custom Styling

Date field with custom styling, initial value, and update callback.
<field-date name="custom-date" placeholder="Enter date" value="2020-01-01" update={(date) => console.log('Updated date:', date)} class="rounded b-solid b-primary p-5 w-200" />
