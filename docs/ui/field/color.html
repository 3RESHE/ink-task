Props

Property Type Required Notes autocomplete String No Sets autocomplete behavior (e.g., "off"). Synced to both inputs. disabled Boolean No If true, disables the color picker and text input. name String No Name attribute for form submission. readonly Boolean No If true, makes both inputs read-only. required Boolean No If true, marks both inputs as required for form validation. value String No Initial color value (e.g., "#ff0000"). Synced between picker and text input. display String No Sets display style of the host (e.g., "inline-block", "block"). Defaults to "inline-block". class String No Custom CSS classes for the host element (e.g., "w-200"). style String No Inline styles for the host element (e.g., "border: 2px solid blue").
