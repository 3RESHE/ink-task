Markdown

import Markdown from '@stackpress/ink-ui/field/markdown';

Props

The `<field-markdown>` component provides a Markdown editor with edit and preview modes. Below are its props and their roles:

Property Type Required Description name String No Name attribute for form submission, tied to a hidden textarea. value String No Initial Markdown content to display in the editor. numbers Boolean No If true, displays line numbers in the editor for better navigation. change Function No Handler called with the change event when content is edited. update Function No Callback invoked with the current Markdown content on change. class String No Custom CSS classes applied to the host element. style String No Inline styles applied to the host element (e.g., "min-height: 200px").

Basic Markdown Editor

A minimal Markdown editor with a form name, featuring edit and preview modes toggled via hover buttons.
<field-markdown name="markdown" />

Markdown with Initial Value

Loads the editor with initial Markdown content using the `value` prop, ready for editing or previewing.
<field-markdown name="markdown" value="# Hello **Bold** text" />

Markdown with Line Numbers

Adds line numbers to the editor with the `numbers` prop, aiding in longer Markdown documents.
<field-markdown name="markdown" numbers value="# Title Line 1 Line 2" />

Markdown with Change Event

Logs changes via the `change` prop, capturing editor input for real-time tracking or validation.
<script> const handleChange = (e) => console.log('Change event:', e.target.value); </script> <field-markdown name="markdown" change={handleChange} value="Edit me" />

Markdown with Update Callback

Uses the `update` prop to track Markdown content changes, ideal for syncing with external state.
<script> const handleUpdate = (value) => console.log('Updated Markdown:', value); </script> <field-markdown name="markdown" update={handleUpdate} value="Update me" />

Styled Markdown Editor

A minimalist styled editor with a wide layout, soft border, and hover effect, enhancing usability and aesthetics.
<field-markdown name="markdown" class="w-400 bg-white tx-t-1" style="border: 1px solid #e0e0e0; padding: 10px; transition: border-color 0.3s ease; min-height: 200px;" onmouseover="this.style.borderColor='#888888'" onmouseout="this.style.borderColor='#e0e0e0'" value="# Styled Markdown" />
