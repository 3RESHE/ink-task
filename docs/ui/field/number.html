Number

import Number from '@stackpress/ink-ui/field/number';

Props

The `<field-number>` component enhances a number input with formatting and validation. Below are its props:

Property Type Required Description name String No Name attribute for form submission. value String/Number No Initial value of the number input (numeric only). min String/Number No Minimum allowed value (numeric only). max String/Number No Maximum allowed value (numeric only). step String/Number No Step interval for increment/decrement (numeric). separator String No Thousands separator (defaults to ","). decimal String No Decimal separator (defaults to "."). absolute Boolean No Forces absolute values (defaults to false). change Function No Custom handler for change events (numeric only). update Function No Callback with the formatted numeric value. background String No Background color for the host element (prefer `bg-*` utilities). border String No Border color for the host element (prefer `b-*` utilities). error Boolean/String No Toggles error state (Boolean) or sets error message (String) for styling/validation. class String No CSS classes for the host element (use Ink utilities). style String No Inline styles (prefer Ink utilities instead). autocomplete String No Controls browser autofill behavior. disabled Boolean No Disables the input. pattern String No Regex pattern for HTML5 validation (numeric). placeholder String No Placeholder text for the input. readonly Boolean No Makes the input read-only. required Boolean No Marks the input as required. type String No Input type (defaults to "number" for numeric input only).

Basic Number Input

A simple number input accepting only numeric values.
<field-number name="quantity" value="1234" />

Number with Range

A number input with `min`, `max`, and `step` constraints, accepting only numeric values.
<field-number name="price" value="50" min="0" max="100" step="5" />

Formatted Number

A number input with custom `separator` and `decimal` formatting, accepting only numeric values.
<field-number name="amount" value="1234567.89" separator="." decimal="," />

Styled Number Input

A styled number input with error state, background, border, and Ink utilities.
<field-number name="quantity" value="1234" error class="w-200 bg-muted b-primary c-5 p-5 transition-300" style="box-shadow: 0 2px 6px var(--t-1);" onmouseover="this.style.boxShadow='0 4px 12px var(--t-3)'" onmouseout="this.style.boxShadow='0 2px 6px var(--t-1)'" />
