Basic Usage

A simple number input with constraints.
<field-number name="basic-number" min="0" max="100" step="1" value="50" />

Formatting Options

Number field with custom separator and decimal formatting.
<field-number name="format-number" min="0" max="10000" step="0.01" value="1234.56" separator=" " decimal="," absolute={true} />

Custom Styling

Number field with custom styling and update callback.
<field-number name="custom-number" min="0" max="10000" step="0.01" value="1234.56" class="w-200 rounded b-solid b-primary p-5" background="var(--gray-light)" update={(value) => console.log('Updated to:', value)} />
