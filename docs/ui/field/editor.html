Editor

import Editor from '@stackpress/ink-ui/field/editor';

Props

Property Type Required Notes name String No Name attribute for form submission, passed to textarea. lang String No Editor mode: "htmlmixed" (default), "markdown", "css", "javascript", "json", "typescript". numbers Boolean No If present, enables line numbers in the editor. value String No Initial content for the editor; overrides child content if set. disabled Boolean No If present, disables the textarea and editor. readonly Boolean No If present, makes the textarea and editor read-only. change Function No Handler called with change event when content updates. update Function No Callback with current editor content on change. class String No CSS classes applied to the host element. style String No Inline styles applied to the host element.

Basic Editor (name)

A basic editor with a form name, using default "htmlmixed" mode.
<field-editor name="editor" />

Editor with Value

An editor with initial HTML content.
<field-editor name="editor" value="<p>Hello, world!</p>" />

Editor with Lang

An editor in Markdown mode with initial content.
<field-editor name="editor" lang="markdown" value="# Title Text" />

Editor with Numbers

An editor with line numbers enabled.
<field-editor name="editor" numbers value="Line 1 Line 2" />

Editor with Change

An editor with a custom change handler.
<script> const handleChange = (e) => console.log('Change event:', e.target.value); </script> <field-editor name="editor" change={handleChange} value="<p>Edit me</p>" />

Editor with Update

An editor with an update callback.
<script> const handleUpdate = (value) => console.log('Updated content:', value); </script> <field-editor name="editor" update={handleUpdate} value="<p>Update me</p>" />

Styled Editor (class, style)

An editor with custom class and style.
<field-editor name="editor" value="<p>Styled</p>" class="w-300" style="border: 1px solid #007bff;" />
