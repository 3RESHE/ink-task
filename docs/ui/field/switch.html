Switch

import Switch from '@stackpress/ink-ui/field/switch';

Props

The `<field-switch>` component provides a toggle switch with customizable styles and labels. Below are its props:

Property Type Required Description name String No Name attribute for form submission. checked Boolean No Initial checked state of the switch (default false). disabled Boolean No Disables the switch. readonly Boolean No Makes the switch read-only. required Boolean No Marks the switch as required. value String No Value submitted when checked (default "on"). label String No Text label displayed next to the switch. onoff Boolean No Displays "ON" and "OFF" labels. yesno Boolean No Displays "YES" and "NO" labels. checkex Boolean No Displays check (✓) and cross (✖) symbols (default true). sunmoon Boolean No Displays sun (☼) and moon (☽) symbols. rounded Boolean No Rounds the switch track and knob. ridge Boolean No Adds a ridged pattern ("|||") to the knob. smooth Boolean No Makes the knob smooth (empty content). blue Boolean No Applies blue theme colors. orange Boolean No Applies orange theme colors. green Boolean No Applies green theme colors. change Function No Custom handler for change events, receiving the event. update Function No Callback with the updated checked state (true/false). class String No CSS classes for the host element (use Ink utilities). style String No Inline styles (prefer Ink utilities instead).

Basic Switch Input

A basic switch input with default check/cross symbols.
<field-switch name="toggle" label="Toggle Me" change={(e) => console.log('Change event:', e.target.checked)} />

Themed Switch Input

A themed switch input with "ON/OFF" labels and green color scheme.
<field-switch name="themeToggle" label="Theme" onoff green checked change={(e) => console.log('Change event:', e.target.checked)} />

Styled Switch Input

A styled switch with sun/moon symbols, rounded corners, and hover effects.
<field-switch name="modeSwitch" label="Mode" sunmoon rounded blue class="tx-lg tx-t-1 transition-300 hover:scale-105" style="cursor: pointer;" change={(e) => console.log('Change event:', e.target.checked)} update={(value) => console.log('Updated value:', value)} />

Yes/No Switch with Orange Theme

A switch with "YES/NO" labels, orange theme, and ridged knob.
<field-switch name="yesNoSwitch" label=" Agree?" yesno orange ridge change={(e) => console.log('Change event:', e.target.checked)} />

Smooth Sun/Moon Switch (Disabled)

A disabled switch with sun/moon symbols and a smooth knob.
<field-switch name="disabledSwitch" label="Day/Night" sunmoon smooth disabled checked class="tx-muted" />

Custom Labeled Switch

A switch with custom label, check/cross symbols, custom value, and hover effects.
<field-switch name="customSwitch" label="Enable Feature" value="enabled" checkex blue checked class="tx-lg tx-t-2 transition-300 hover:shadow-0-2-8-t-3" style="cursor: pointer;" change={(e) => console.log('Change event:', e.target.checked)} update={(value) => console.log('Updated value:', value)} />
