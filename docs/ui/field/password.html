Password

import Password from '@stackpress/ink-ui/field/password';

Props

The `<field-password>` component enhances a password input with a toggle to show/hide the password. Below are its props:

Property Type Required Description name String No Name attribute for form submission. value String No Initial value of the password input. autocomplete String No Controls browser autofill behavior (e.g., "off"). disabled Boolean No Disables the input. pattern String No Regex pattern for HTML5 validation. placeholder String No Placeholder text for the input. readonly Boolean No Makes the input read-only. required Boolean No Marks the input as required. change Function No Custom handler for change events. update Function No Callback with the updated password value. class String No CSS classes for the host element (use Ink utilities). style String No Inline styles (prefer Ink utilities instead).

Basic Password Input

A basic password input with show/hide toggle.
<field-password name="password" value="secret123" />

Password with Attributes

A password input with additional attributes like `required` and `autocomplete`.
<field-password name="password" value="secret123" required autocomplete="off" />

Styled Password Input

A styled password input with enhanced toggle appearance using Ink utilities.
<field-password name="password" value="secret123" class="w-200 bg-secondary b-primary c-5 p-5 transition-300 tx-muted hover:tx-primary" style="box-shadow: 0 2px 6px var(--t-1);" onmouseover="this.style.boxShadow='0 4px 12px var(--t-3)'" onmouseout="this.style.boxShadow='0 2px 6px var(--t-1)'" />
