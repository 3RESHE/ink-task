Slug

import Slug from '@stackpress/ink-ui/field/slug';

Props

The `<field-slug>` component provides a text input that transforms input into slugs (with dashes or underscores) or camelCase. Below are its props:

Property Type Required Description name String No Name attribute for form submission. value String No Initial value of the input (transformed on change). placeholder String No Placeholder text when input is empty. autocomplete String No Controls browser autofill behavior. disabled Boolean No Disables the input. pattern String No Regular expression pattern for validation (e.g., "^[a-z0-9-]+$"). readonly Boolean No Makes the input read-only. required Boolean No Marks the input as required. dash Boolean No Uses dashes (`-`) for word separation (default true, overrides `line` and `camel`). line Boolean No Uses underscores (`_`) for word separation (default false, overrides `dash` if true). camel Boolean No Converts to camelCase (default false, overrides `dash` and `line` if true). change Function No Custom handler for change events, receiving the event. update Function No Callback with the transformed value. class String No CSS classes for the host element (use Ink utilities). style String No Inline styles (prefer Ink utilities instead).

Basic Slug Input

A basic slug input with default dash separation.
<field-slug name="slug" value="Hello World" update={(value) => console.log('Transformed:', value)} />

CamelCase Slug Input

A slug input transformed to camelCase.
<field-slug name="camelSlug" value="Hello World" camel update={(value) => console.log('Transformed:', value)} />

Styled Slug Input

A styled slug input with custom formatting and hover effects.
<field-slug name="styledSlug" value="Hello World Today" dash placeholder="Enter slug text" pattern="^[a-z0-9-]+$" class="w-300 p-5 b-solid b-t-2 c-5 tx-lg tx-t-1 bg-white transition-300 hover:b-primary hover:shadow-0-2-8-t-3" style="cursor: text;" update={(value) => console.log('Transformed:', value)} />
