Filelist Field

import FilelistField from '@stackpress/ink-ui/field/filelist';

Props

Name Type Required Notes name String No Name attribute for form submission value Array No Initial array of file URLs (default: []) image Boolean No Restricts to images and shows previews uploading String No Text during upload (default: "Uploading...") upload Function No Upload handler (files, next) => void; next receives URL array change Function No Callback for change event (receives ChangeEvent) update Function No Callback with updated URL array (not implemented in source)

Basic Usage

A simple filelist field without upload handling.
<field-filelist name="basic-filelist" class="block w-250" />

Image Mode

Filelist field restricted to images, showing previews of uploaded files after a mock upload (uses a static URL for demo; replace upload function for real URLs).
<script> const filelistupload = (files, next) => { setTimeout(() => { next(files.map((file, i) => 'https://images.wsj.net/im-580612/8SR')); }, 1000); }; </script> <field-filelist name="image-filelist" image={true} upload={filelistupload} class="block w-250" />

Default Values

Filelist field with preloaded image URLs and removal options.
<field-filelist name="default-filelist" image={true} class="block w-250" value={['https://example.com/sample1.jpg', 'https://example.com/sample2.jpg']} />

Custom Upload

Filelist field with custom upload function and update callback.
<script> const filelistupload = (files, next) => { setTimeout(() => { next(files.map((file, i) => 'https://images.wsj.net/im-580612/8SR')); }, 1000); }; </script> <field-filelist name="custom-filelist" image={true} class="block w-250 rounded b-solid b-primary" upload={filelistupload} update={(urls) => console.log('Uploaded URLs:', urls)} />
