Select

import Select from '@stackpress/ink-ui/field/select';

Props

The `<field-select>` component provides a customizable dropdown select input. Options can be defined using `<option>` or `<span class="select-default-option">` tags. Below are its props:

Property Type Required Description name String No Name attribute for form submission. value Array/String No Initial selected value(s); String for single, Array or JSON string (e.g., "[1, 2]") for multiple. placeholder String No Placeholder text when no options are selected (defaults to "Choose Option"). custom Boolean No Enables custom option input. search Boolean No Enables search functionality within options. multiple Boolean No Enables multiple selection. disabled Boolean No Disables the select input. required Boolean No Marks the input as required. open Function No Callback when the dropdown opens. close Function No Callback when the dropdown closes. filter Function No Callback when the search input changes, receiving the event. select Function No Callback when an option is selected, receiving the event. remove Function No Callback when an option is removed, receiving the event. add Function No Callback when a custom option is added, receiving the event. clear Function No Callback when selections are cleared, receiving the event. change Function No Custom handler for change events, receiving the event. update Function No Callback with the updated value(s). class String No CSS classes for the host element (use Ink utilities). style String No Inline styles (prefer Ink utilities instead).

Basic Select Input

A basic single-select input with predefined options.
<field-select name="color"> <option value="red">Red</option> <option value="blue">Blue</option> <option value="green">Green</option> </field-select>

Select with Search

A select input with search functionality and multiple selection.
<field-select name="fruits" value='["apple", "banana"]' search multiple filter={(e) => console.log('Filter event:', e.target.value)} > <option value="apple">Apple</option> <option value="banana">Banana</option> <option value="orange">Orange</option> <option value="grape">Grape</option> <option value="pineapple">Pineapple</option> <option value="mango">Mango</option> </field-select>

Enhanced Select Input

An enhanced select input with multiple features and event handlers.
<field-select class="w-200 relative z-1" name="select" placeholder="Choose" value='["1"]' search custom multiple open={(e) => console.log('open', e)} close={(e) => console.log('close', e)} filter={(e) => console.log('filter', e)} select={(e) => console.log('select', e)} remove={(e) => console.log('remove', e)} add={(e) => console.log('add', e)} clear={(e) => console.log('clear', e)} change={(e) => console.log('change', e)} update={(e) => console.log('update', e)} > <option value="1" keyword="option 1">Option 1</option> <option value="4" keyword="option 2"><strong>Option 2</strong></option> <option value="3" keyword="option 3">Option 3</option> <option value="2" keyword="option 4">Option 4</option> </field-select>

Styled Select Input

A styled select input with custom options, styling, and hover effects using Ink utilities.
<field-select name="category" placeholder="Select Category" custom class="w-200 p-5 b-solid b-t-2 c-5 bg-white tx-t-1 transition-300 hover:b-primary hover:shadow-0-2-8-t-3" style="cursor: pointer;" > <option value="tech">Technology</option> <option value="sports">Sports</option> <option value="music">Music</option> </field-select>
