Mask

import Mask from '@stackpress/ink-ui/field/mask';

Props

The `<field-mask>` component enhances inputs with custom masks using the `Inputmask` library. Below are its props for controlling masking and input behavior:

Property Type Required Description name String No Name attribute for form submission, linking the input to form data. value String No Initial value of the masked input (e.g., "1234567890" for a phone mask). mask String/Array No Custom mask pattern (e.g., "(999) 999-9999"). Use "9" for digits, "A" for letters, etc. regex String No Regex pattern for masking (e.g., "^[0-9]{4}$" for 4 digits). Overrides `mask` if set. alias String No Predefined mask alias (e.g., "phone", "date", "email"). Simplifies common formats. repeat Number No Repeats the mask pattern a specified number of times. greedy Boolean No Forces the mask to be fully completed before allowing input to proceed. numericInput Boolean No Fills numeric masks from right to left, ideal for numbers like currency. rightAlign Boolean No Aligns input text to the right within the field. definitions Object No Custom mask character definitions (e.g., { "X": "[A-Z]" }). Extends mask syntax. placeholder String No Hint text shown when the input is empty, guiding user input (e.g., "(123) 456-7890"). autocomplete String No Controls browser autofill behavior (e.g., "off", "tel"). disabled Boolean No Disables the input, preventing interaction. pattern String No Regex pattern for HTML5 validation (e.g., "[0-9]{10}"). Works alongside mask. readonly Boolean No Makes the input read-only, displaying the mask without edits. required Boolean No Marks the input as required for form submission. change Function No Custom handler for change events, receiving the event object. update Function No Callback with the current masked value after changes. error Function No Callback to handle errors, receiving an error message when validation fails. class String No CSS classes for the host element, styled via default utilities. style String No Inline styles for the host element, overriding defaults.

Basic Phone Mask

A simple phone number mask using the `mask` prop, formatting input as "(999) 999-9999".
<field-mask name="phone" mask="(999) 999-9999" />

Alias Mask (Date)

Uses the `alias` prop for a predefined date mask, formatting as "mm/dd/yyyy".
<field-mask name="date" alias="datetime" mask="mm/dd/yyyy" />

Regex Mask

Applies a regex mask with `regex`, restricting input to exactly 4 digits (e.g., "1234").
<field-mask name="code" regex="^[0-9]{4}$" />

Numeric Input Mask

A numeric mask with `numericInput` for right-to-left entry, ideal for currency or codes.
<field-mask name="amount" mask="9999" numericInput />

Mask with Placeholder

Adds a `placeholder` to guide the user on the expected format of the phone mask.
<field-mask name="phone" mask="(999) 999-9999" placeholder="(123) 456-7890" />

Mask with Validation

Combines a phone mask with `required` and `pattern` for form validation, ensuring the input matches the mask format.
<field-mask name="phone" mask="(999) 999-9999" required pattern="([0-9]{3}) [0-9]{3}-[0-9]{4}" />

Mask with Change Event

Logs changes with the `change` prop, capturing masked input for real-time feedback.
<script> const handleChange = (e) => console.log('Change event:', e.target.value); </script> <field-mask name="phone" mask="(999) 999-9999" change={handleChange} />

Mask with Update Callback

Tracks masked value changes with `update`, syncing with external state.
<script> const handleUpdate = (value) => console.log('Updated value:', value); </script> <field-mask name="phone" mask="(999) 999-9999" update={handleUpdate} />

Mask with Error Handling

Uses the `error` prop to handle invalid input by validating the phone number format on change.
<field-mask name="phone" mask="(999) 999-9999" change={(e) => { const value = e.target.value; const isValid = /^(d{3}) d{3}-d{4}$/.test(value); if (!isValid && value.length === 14) { e.target.error('Invalid phone number format. Expected: (123) 456-7890'); } }} error={(message) => console.log('Error:', message)} />

Styled Mask

A minimalist styled mask with a wide layout, soft border, and hover effect via `class` and `style`.
<field-mask name="phone" mask="(999) 999-9999" class="w-300 bg-primary tx-t-1" style="border: 1px solid #e0e0e0; padding: 8px; transition: border-color 0.3s ease;" onmouseover="this.style.borderColor='#888888'" onmouseout="this.style.borderColor='#e0e0e0'" />
