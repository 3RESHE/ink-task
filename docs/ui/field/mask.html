Mask Field

import MaskField from '@stackpress/ink-ui/field/mask';

Props

Basic Usage

A simple masked input for a phone number.
<field-mask name="phone" mask="999-999-9999" placeholder="999-999-9999" />

Custom Mask

Masked input with a custom date format and numeric input.
<field-mask name="date" mask="99/99/9999" placeholder="MM/DD/YYYY" numericInput={true} />

Alias Mask

Masked input using a predefined alias for email.
<field-mask name="email" alias="email" placeholder="user@domain.com" />

Numeric Input

Masked input restricted to numbers with a disabled state.
<field-mask name="numeric" mask="9999-9999" numericInput={true} placeholder="1234-5678" disabled={true} />

Right Alignment

Masked input aligned to the right with a currency mask.
<field-mask name="currency" mask="$ 999.99" placeholder="$ 123.45" rightAlign={true} numericInput={true} />

Repeated Mask

Masked input with a repeating pattern and greedy behavior.
<field-mask name="repeat" mask="9" repeat={5} greedy={true} placeholder="12345" />

Styling

Masked input with custom styling and change callback.
<field-mask name="custom-phone" mask="(999) 999-9999" placeholder="(999) 999-9999" class="w-200 rounded b-solid b-primary p-5" change={(e) => console.log('Changed to:', e.target.value)} />
