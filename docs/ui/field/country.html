Country

import Country from '@stackpress/ink-ui/field/country';

Props

Property Type Required Notes name String No Name attribute for hidden inputs in form submission. value String No Initial selected country code (e.g., "US"). Single selection only. placeholder String No Text shown when no country is selected. Defaults to "Select Country". class String No Custom CSS classes for the host element (e.g., "w-200"). style String No Inline styles for the host element (e.g., "border: 2px solid blue").

Basic Country

A simple country selector with default placeholder.
<field-country name="country" class="relative z-1" />

Country with Value

A country selector with an initial selected value.
<field-country name="country" value="US" class="relative z-1" />

Custom Placeholder

A country selector with a custom placeholder text.
<field-country name="country" placeholder="Choose a Country" class="relative z-1" />

Styled Country

A country selector with custom width and styling.
<field-country name="country" value="CA" class="w-200 relative z-1" style="border-color: var(--primary);" />
