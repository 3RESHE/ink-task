Form Control

import Control from '@stackpress/ink-ui/form/control';

Props

Name Type Required Notes label String No Text for the field label error String No Error message; sets text color to var(--error) if present

Basic Usage

A simple form control with a label and input.
<form-control label="First Name"> <field-input name="first" placeholder="Enter your first name" /> </form-control>

Error State

Display an error message below the field.
<form-control class="py-5" label="First Name*" error="Some Error"> <field-input name="first" placeholder="Enter your first name" error /> </form-control>

Combined Fields

Multiple controls in a form with varying states.
<form-control class="py-5" label="First Name*"> <field-input name="first" placeholder="Enter your first name" /> </form-control> <form-control class="py-5" label="Last Name*" error="Last name is required"> <field-input name="last" placeholder="Enter your last name" error /> </form-control> <form-control class="py-5" label="Email"> <field-input name="email" placeholder="Enter your email" type="email" /> </form-control>
