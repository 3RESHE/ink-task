Control

import Control from '@stackpress/ink-ui/form/control';

Props

Property Type Required Notes label String No Text for the label above the slotted content. Omitted if empty or falsy. error String No Error message below the slotted content. Shown only if non-empty; sets host color to var(--error) (red). class String No Custom CSS classes applied to the host element (e.g., "py-5 tx-bold"). style String No Inline styles applied to the host element (e.g., "border: 1px solid red").

Basic Control

A simple control with a label and an input field.
<form-control label="Email"> <field-input name="email" placeholder="Enter your email" /> </form-control>

Control with Error

A control displaying an error message below the input.
<form-control class="py-5" label="First Name*" error="Some Error"> <field-input name="first" placeholder="Enter your first name" error /> </form-control>

Control without Label

A control with no label, only an input and optional error.
<form-control error="Required"> <field-input name="phone" placeholder="Enter phone number" /> </form-control>

Control with Checkbox

A control wrapping a checkbox with a label and error feedback.
<form-control label="Agree to Terms" error="Must agree"> <field-checkbox name="terms" value="yes" /> </form-control>

Control with Select

A control with a dropdown select input.
<form-control label="Country" class="w-200"> <field-select name="country"> <option value="">Select a country</option> <option value="us">United States</option> <option value="ca">Canada</option> </field-select> </form-control>
