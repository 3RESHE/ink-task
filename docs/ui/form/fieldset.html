Fieldset

import Fieldset from '@stackpress/ink-ui/form/fieldset';

Props

Property Type Required Notes multiple Boolean No If true, allows multiple rows with an "Add" button. Inputs become an empty array if undefined. inputs Object | Array<Object> No Data for each row. Single object if multiple=false, array if multiple=true. Defaults to [{}] if invalid. errors Object | Array<Object> No Error data per row. Matches inputs structure. Defaults to [{}] if invalid. add String No Text for the "Add" button when multiple=true. Defaults to "Add". display String No Sets display style of the host (e.g., "block", "inline"). Defaults to "block". border String No Applies border styles (e.g., "solid", "dashed"). Handled by Ink UI utilities. radius String No Sets border radius (e.g., "md", "lg"). Handled by Ink UI utilities. button String No Styles the "Add" button (e.g., "primary", "success"). Requires multiple=true. class String No Custom CSS classes for the host element (e.g., "p-10"). style String No Inline styles for the host element (e.g., "margin: 10px").

Multiple Fieldset

A fieldset with multiple rows and an "Add" button.
<script> const sampleData = [ { first: 'John', last: 'Doe' }, { first: 'Jane', last: 'Smith' } ]; </script> <form-fieldset legend="Users" multiple inputs={sampleData} add="Add User"> <field-input name="first" placeholder="First Name" /> <field-input name="last" placeholder="Last Name" /> </form-fieldset>

Styled Fieldset

A multiple fieldset with an "Add" button to append single-input rows.
<form-fieldset legend="Contacts" multiple class="p-10" border="solid" radius="md" add="New Contact" > <field-input name="email" placeholder="Email Address" class="w-full" /> </form-fieldset>
