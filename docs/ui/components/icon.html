Icons

Icons

import Icon from '@stackpress/ink-ui/element/icon';

Props

Name Type Required Notes name String Yes Font Awesome icon name (e.g., "home", "github", "bell") brand Boolean No Uses brand style for logos (e.g., "github", "twitter"); defaults to solid style otherwise size Number No Custom font size in pixels (e.g., 16, 32) xs, sm, md, lg, xl, xl2, xl3, xl4, xl5 Boolean No Predefined size variants (escalating from extra-small to extra-large) color String No Custom CSS color (e.g., "red", "#facc15", "info")

Types

Icons can be solid (default) or brand-specific (e.g., logos):
<element-icon name="user" />
<element-icon name="steam" brand />
<element-icon name="github" brand />

Size

Control icon size with custom values or predefined variants:
<element-icon name="bell" size="16" />
<element-icon name="bell" size="32" />
<element-icon name="bell" lg />
<element-icon name="bell" sm />

Color

Apply custom colors to icons:
<element-icon name="heart" color="red" />
<element-icon name="star" color="#facc15" />
<element-icon name="info-circle" info />
