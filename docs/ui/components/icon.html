Basic Usage

Basic icons with default solid style.
<element-icon name="star" /> <element-icon name="heart" /> <element-icon name="bell" />

Colors

Customize icon colors with predefined or custom values.
<element-icon name="info-circle" info /> <element-icon name="exclamation-triangle" warning /> <element-icon name="times-circle" error /> <element-icon name="check-circle" success />
<element-icon name="star" color="purple" /> <element-icon name="heart" color="#ff4500" /> <element-icon name="bell" primary />

Sizes

Adjust icon sizes with predefined options.
<element-icon name="star" sm /> <element-icon name="star" md /> <element-icon name="star" lg /> <element-icon name="star" xl2 />

Types

Use solid or brand icon styles.
<element-icon name="star" solid /> <element-icon name="twitter" brand /> <element-icon name="github" brand />
