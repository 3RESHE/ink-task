Tab

import Tab from '@stackpress/ink-ui/element/tab';

Props

Property Type Required Notes group String Yes Groups tabs together. Only one tab in a group can be active at a time. Must be unique per tab set (e.g., "http", "user"). selector String No CSS selector for content to show/hide when this tab is active (e.g., "#user-profile"). Leave empty if no content is controlled. active String No Space-separated class names applied when the tab is active (e.g., "bg-black tx-white"). Use atomic styles for customization. inactive String No Space-separated class names applied when the tab is inactive (e.g., "bg-t-1 tx-muted"). Contrasts with `active` for visual feedback. style String No Inline CSS for the tab’s `<a>` element (e.g., "padding: 12px;"). Overrides default styles like `height: 100%`. class String No Additional class names for the `<a>` element. Complements `active`/`inactive` classes. on Boolean No Sets the tab as active on mount. Only one tab per group should have `on`. Use as an attribute (e.g., `on=""`).
