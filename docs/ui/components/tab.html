Tab

import Tab from '@stackpress/ink-ui/element/tab';

Props

Property Type Required Notes group String Yes Defines the group of tabs this tab belongs to. selector String No The selector used to show or hide content based on the tab selected. active String No Class names to apply when the tab is active. inactive String No Class names to apply when the tab is inactive. style String No Inline style for custom styling of the tab. class String No CSS class name(s) to apply to the tab element.
