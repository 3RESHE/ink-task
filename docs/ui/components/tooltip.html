Tooltips

import Tooltip from '@stackpress/ink-ui/element/tooltip';

Props

Property Type Required Notes top Boolean/String/Number No Positions the tooltip at the top. Can be a boolean, number (px), or string (e.g., "10px"). bottom Boolean/String/Number No Positions the tooltip at the bottom. Can be a boolean, number (px), or string (e.g., "10px"). left Boolean/String/Number No Positions the tooltip to the left. Can be a boolean, number (px), or string (e.g., "10px"). right Boolean/String/Number No Positions the tooltip to the right. Can be a boolean, number (px), or string (e.g., "10px"). color String No Sets the tooltip text color (e.g., "info", "primary", "success"). white Boolean No Sets the tooltip text to white. black Boolean No Sets the tooltip text to black. info Boolean No Sets the tooltip text to an informational color (blue). warning Boolean No Sets the tooltip text to a warning color (yellow). success Boolean No Sets the tooltip text to a success color (green). error Boolean No Sets the tooltip text to an error color (red). muted Boolean No Sets the tooltip text to a muted color (gray). primary Boolean No Sets the tooltip text to a primary color (brand color). secondary Boolean No Sets the tooltip text to a secondary color. curved Boolean No Applies a curved border to the tooltip. curve Number No Specifies the radius of the tooltip's border. center Boolean No Centers the tooltip text. padding Number No Sets the padding inside the tooltip (in pixels). opacity Number No Adjusts the opacity of the tooltip (0-100). width String/Number No Sets the width of the tooltip. If "auto", it adjusts automatically. Can also be a number (px). background String No Sets the background color of the tooltip.

Tooltip with Info Background

This example demonstrates a tooltip with an info background. It appears above the text with curved corners, some padding, and an opacity of 80%. The tooltip shows when you hover over the "Hover me!" text, positioned above it.
Hover me! This is the first and last name
<element-tip background-info curved top="-30" left="50" padding="5" opacity="80" width="150" > This is the first and last name </element-tip>

Tooltip with Success Background

This example shows a tooltip with a success background (green). The tooltip is positioned to the bottom-right of the element, with a subtle curve and padding.
Hover me to see success tooltip! Successfully completed!
<element-tip background-success curved bottom="10" right="10" padding="10" opacity="90" > Successfully completed! </element-tip>

Tooltip with Error Background

This example demonstrates a tooltip with an error background (red) that appears on hover. The tooltip has custom width, and the text is displayed with some padding and a background color. It appears above the text and is aligned centrally.
Hover me to see error tooltip! An error has occurred.
<element-tip background-error curved top="20" left="50" padding="8" opacity="95" width="180" > An error has occurred. </element-tip>

Tooltip with Warning Background

This example features a tooltip with a warning background (yellow). It appears to the right of the trigger element and has a larger font size for better visibility.
Hover me to see warning tooltip! Please take caution!
<element-tip background-warning curved right="20" padding="12" opacity="85" width="200" > Please take caution! </element-tip>

Tooltip with Centered Text and Curved Border

This example demonstrates a tooltip with a centered alignment, and the tooltip has a curved border. The tooltip will be displayed above the text, with subtle styling.
Hover over me! Information about this text.
<element-tip background-muted curved top="-20" padding="8" opacity="80" width="120" center > Information about this text. </element-tip>

Tooltip with Custom Curve Radius

This example shows a tooltip with a custom border radius (10px) using the `curve` prop and a primary background. It appears below the trigger text.
Hover me for custom curve! Custom radius tooltip!
<element-tip background-primary curve="10" bottom="10" padding="10" opacity="90" width="160" > Custom radius tooltip! </element-tip>
