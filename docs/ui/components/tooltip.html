Basic Usage

A simple tooltip that appears on hover.
Hover me Tooltip text
<span class="relative inline-block p-10"> Hover me <element-tip background={ { muted: true } } top="-30" left="0" padding="5">Tooltip text</element-tip> </span>

Position

Adjust tooltip position with top, bottom, left, or right.
