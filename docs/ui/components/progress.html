Progress

import Badge from '@stackpress/ink-ui/element/progress';

Props

Name Type Required Notes width Number Yes Defines the width of the progress bar, from 0 to 100%. height Number No Sets the height of the progress bar (default: 20px). bg String No Sets the background color of the progress container. children Slot No Allows custom label text to be displayed within the progress bar.

Progress Bar Examples

This example demonstrates a simple progress bar with a 50% width, "info" color, and a muted background. It has rounded corners and a height of 20px for a clean, minimal look.
<element-progress width={50} info class="bg-muted rounded tx-13 h-20 tx-lh-20 w-full-20" />
This example demonstrates a progress bar with 75% width, the "success" color, and a background of primary color. The height is set to 30px, giving it a taller appearance.
<element-progress width={75} success class="bg-primary rounded tx-13 h-30 tx-lh-30 w-full-30" />
This example demonstrates a progress bar with a width of 40%, the "warning" color, and a height of 10px for a thinner, more compact look.
<element-progress width={40} warning class="bg-warning rounded tx-13 h-10 tx-lh-10 w-full-10" />
This example features a 100% width progress bar with the "success" color, a height of 20px, and a full-length progress bar to indicate completion.
<element-progress width={100} success class="bg-success rounded tx-13 h-20 tx-lh-20 w-full-20" />
This example demonstrates a progress bar with a 20% width, a height of 10px, and an "error" background color to indicate a negative status or issue. The "error" color highlights the progress bar as a warning or error state.
<element-progress width={20} class="rounded bg-error tx-13 h-10 tx-lh-10 w-full-10" />
