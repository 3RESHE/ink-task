Breadcrumbs

import Crumbs from '@stackpress/ink-ui/element/crumbs';

Props

Name Type Required Notes crumbs Array Yes List of breadcrumb items with labels, icons, and links block Boolean No Display breadcrumbs as a block element link Object No Styles and properties for breadcrumb links sep Object No Styles and properties for separators icon Object No Styles and properties for icons xs, sm, md, lg, xl, xl2, xl3, xl4, xl5 Boolean No Size variations for icons and separators color String No Color for icons and separators white, black, info, warning, success, error, muted, primary, secondary Boolean No Color variations for icons and separators spacing Number No Spacing between separator icons

Basic Usage

Responsive breadcrumb layout inside a container:
<script> const crumbs = [ { icon: 'home', label: 'Home', href: '/ink/index.html' }, { icon: 'book', label: 'Docs', href: '/ink/docs/index.html' }, { icon: 'icons', label: 'UI', href: '/ink/ui/index.html' }, { icon: 'icons', label: 'Components', href: '/ink/ui/index.html' }, { label: 'Crumbs' } ]; </script> <div class="basis-third-10 lg-basis-half-10 md-basis-full"> <div class="bg-t-3 h-120 flex flex-center p-10 rounded-2xl shadow-md"> <element-crumbs crumbs={crumbs} block bold white underline icon-muted link-primary spacing={2} /> </div> <a class="block tx-center tx-primary p-10 b-solid b-t-3 b-1 hover-bg-t-2 rounded-b-2xl transition-all" href="/ink/ui/components/crumbs.html" > Crumbs </a> </div>

Link Styling

Example with colored link:
<element-crumbs crumbs={crumbs} link-warning block />
<element-crumbs crumbs={crumbs} link-success block />

Separator Variants

Example with muted separator:
<element-crumbs crumbs={crumbs} sep-muted link-primary />
Example with colored separator:
<element-crumbs crumbs={crumbs} sep-success link-primary />
<element-crumbs crumbs={crumbs} sep-warning link-primary />

Icon Styling

Example with muted icons:
<element-crumbs crumbs={crumbs} icon-muted link-primary block />
Example with colored icons:
<element-crumbs crumbs={crumbs} icon-success link-primary block />
<element-crumbs crumbs={crumbs} icon-warning link-primary block />

Size Variants

Example with different sizes:

<element-crumbs crumbs={crumbs} link-primary block xs/>

Spacing & Customization

<element-crumbs crumbs={crumbs} spacing={2} link-primary block />
Example with custom colors:
<element-crumbs crumbs={crumbs} color="white" link-primary block />

<element-crumbs crumbs={crumbs} link={{color: 'salmon'}} block />
