Breadcrumbs

import Crumbs from '@stackpress/ink-ui/element/crumbs';

Props

Name Type Required Notes crumbs Array Yes Array of objects with "icon" (string), "label" (string), and "href" (string) properties block Boolean No Displays crumbs as a block element (default) flex Boolean No Displays crumbs as a flex container with centered items bold Boolean No Applies bold styling to non-link labels underline Boolean No Underlines all links link Object No Custom styles for links (e.g., { color: "salmon" }) sep Object No Custom styles for separator icons (e.g., { color: "var(--muted)" }) icon Object No Custom styles for item icons (e.g., { color: "var(--success)" }) size Number No Custom font size in pixels for crumbs xs, sm, md, lg, xl, xl2, xl3, xl4, xl5 Boolean No Predefined size variants for crumbs (escalating from extra-small to extra-large) color String No Custom CSS color for all text/icons (e.g., "salmon", "#ff0000") white, black, info, warning, success, error, muted, primary, secondary Boolean No Predefined color variants for all text/icons spacing Number No Horizontal spacing between items in pixels (default: 0)

Basic Usage

A simple breadcrumb layout with icons and links:
<script> const crumbs = [ { icon: 'home', label: 'Home', href: '/ink/index.html' }, { icon: 'book', label: 'Docs', href: '/ink/docs/index.html' }, { icon: 'icons', label: 'UI', href: '/ink/ui/index.html' }, { icon: 'icons', label: 'Components', href: '/ink/ui/index.html' }, { label: 'Crumbs' } ]; </script> <element-crumbs crumbs={crumbs} block bold white sep-muted link-primary spacing={2} />

Link Styling

Customize link colors using predefined variants or custom values:
<element-crumbs crumbs={crumbs} link-warning block underline />
<element-crumbs crumbs={crumbs} link={{ color: 'salmon' }} block />

Separator Variants

Adjust separator appearance with colors:
<element-crumbs crumbs={crumbs} sep-muted link-primary block />
<element-crumbs crumbs={crumbs} sep-success link-primary block />

Icon Styling

Style breadcrumb icons with colors:
<element-crumbs crumbs={crumbs} icon-muted link-primary block />
<element-crumbs crumbs={crumbs} icon-success link-primary block />

Size Variants

Adjust crumb size with predefined variants:
<element-crumbs crumbs={crumbs} xs link-primary block />
<element-crumbs crumbs={crumbs} lg link-primary block />

Spacing & Customization

Customize spacing between items:
<element-crumbs crumbs={crumbs} spacing={5} link-primary block />
<element-crumbs crumbs={crumbs} spacing={2} link-primary block />
Apply custom colors to all elements:
<element-crumbs crumbs={crumbs} color="salmon" block />
