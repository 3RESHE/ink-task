Alert

import Alert from '@stackpress/ink-ui/element/alert';

Props

Name Type Required Notes outline Boolean No Displays the alert with an outlined style solid Boolean No Displays the alert with a solid background transparent Boolean No Displays the alert with a transparent background padding Number No Custom padding for the alert (default: 16px) color String No Custom CSS-compatible color (e.g., "salmon", "#ff0000") info, warning, success, error, muted Boolean No Predefined color variations white, black, primary, secondary Boolean No Additional predefined colors curved Boolean No Applies a slightly rounded border radius rounded Boolean No Applies a fully rounded border radius pill Boolean No Applies a pill-shaped border radius

Types

Alerts have the following types: info, warning, success, error, and muted.
No Results found <element-alert info> <element-icon name="info-circle" /> No Results found </element-alert> Are you sure? <element-alert warning> <element-icon name="exclamation-triangle" /> Are you sure? </element-alert> Successfully saved! <element-alert success> <element-icon name="check-circle" /> Successfully saved! </element-alert> Could not save <element-alert error> <element-icon name="exclamation-circle" /> Could not save </element-alert> I am disabled <element-alert muted> I am disabled </element-alert>

Custom Color

Alerts can have custom CSS compatible colors which includes hex and color names.
Who likes salmon? <element-alert color="salmon"> <element-icon name="exclamation-circle" /> Who likes salmon? </element-alert>

Rounded

Alerts can be rounded in three ways: curved, rounded, and pill.
No Results found <element-alert info curved> <element-icon name="info-circle" /> No Results found </element-alert> No Results found <element-alert warning rounded> <element-icon name="exclamation-triangle" /> Are you sure? </element-alert> Successfully saved! <element-alert success pill> <element-icon name="check-circle" /> No Results found </element-alert>

Padding

Alerts can have custom padding values. The default is 16px, but you can override it by setting apadding attribute with a numeric value.
Extra padded success message! <element-alert success padding="32"> <element-icon name="check-circle" /> Extra padded success message! </element-alert>

Transparent

Setting transparent, removes the background color but retains the border.
This is a transparent alert. <element-alert transparent info> <element-icon name="info-circle" /> This is a transparent alert. </element-alert>

Outline Example

An outlined alert has a colored border instead of a filled background.
Warning alert with outline! <element-alert outline warning> <element-icon name="exclamation-triangle" /> Warning alert with outline! </element-alert>

Combining Props

You can combine multiple props like outlined, cruved, and padding to create different styles.
Curved outline with padding. <element-alert outline curved padding="24" muted> <element-icon name="info-circle" /> Curved outline with padding. </element-alert>
