Badge

import Badge from '@stackpress/ink-ui/element/badge';

Props

Name Type Required Notes outline Boolean No Displays the badge with a white background and colored border/text solid Boolean No Displays the badge with a solid colored background (default if outline/transparent omitted) transparent Boolean No Displays the badge with a transparent background and colored border/text padding Number No Sets custom padding in pixels (default: "2px 8px") color String No Sets a custom CSS color for background (solid) or text/border (outline/transparent), e.g., "salmon", "#ff0000" info Boolean No Sets a predefined info style (blue background) warning Boolean No Sets a predefined warning style (yellow background) success Boolean No Sets a predefined success style (green background) error Boolean No Sets a predefined error style (red background) muted Boolean No Sets a predefined muted style (gray background) curved Boolean No Applies a slightly rounded border radius (4px) rounded Boolean No Applies a fully rounded border radius (8px) pill Boolean No Applies a pill-shaped border radius (9999px)

Types

Badges have the following predefined color types: info, warning, success, error, and muted.
100
<element-badge info class="tx-sm">100</element-badge>
200
<element-badge warning class="tx-sm">200</element-badge>
300
<element-badge success class="tx-sm">300</element-badge>
400
<element-badge error class="tx-sm">400</element-badge>
500
<element-badge muted class="tx-sm">500</element-badge>

Custom Color

Badges can use custom CSS-compatible colors for the background (solid) or text/border (outline/transparent).
600
<element-badge color="salmon" class="tx-sm">600</element-badge>
Hex Color
<element-badge color="#ffcc00" class="tx-sm">Hex Color</element-badge>

Rounded

Badges can be rounded in three ways: curved (4px), rounded (8px), or pill (9999px).
100
<element-badge info curved class="tx-sm">100</element-badge>
200
<element-badge warning rounded class="tx-sm">200</element-badge>
300
<element-badge success pill class="tx-sm">300</element-badge>

Padding

The padding of a badge can be adjusted using the padding prop, defaulting to 2px 8px.
Padding 4
<element-badge info padding="4" class="tx-sm">Padding 4</element-badge>
Padding 10
<element-badge success padding="10" class="tx-sm">Padding 10</element-badge>

Border Styles

Badges can be styled with outline or transparent properties to modify their appearance.
Outline
<element-badge outline info class="tx-sm">Outline</element-badge>
Transparent
<element-badge transparent info class="tx-sm">Transparent</element-badge>
