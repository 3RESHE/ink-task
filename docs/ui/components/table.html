Props

Property Type Required Notes nowrap Boolean No Prevents text wrapping inside table cells. wrap1 Boolean No Sets a cell width of 100px for text wrapping. wrap2 Boolean No Sets a cell width of 200px for text wrapping. wrap3 Boolean No Sets a cell width of 300px for text wrapping. wrap4 Boolean No Sets a cell width of 400px for text wrapping. wrap5 Boolean No Sets a cell width of 500px for text wrapping. top String | Number No Makes the table header sticky at the top. Accepts a pixel or percentage value. bottom String | Number No Makes the table footer sticky at the bottom. Accepts a pixel or percentage value. left String | Number No Makes a column sticky on the left side. Accepts a pixel or percentage value. right String | Number No Makes a column sticky on the right side. Accepts a pixel or percentage value. head String No Custom class list for styling the table header. body String No Custom class list for styling the table body. odd String No Custom class list for styling odd table rows. even String No Custom class list for styling even table rows. foot String No Custom class list for styling the table footer. sticky Boolean No Enables sticky positioning for headers, footers, or columns if any sticky props are set.
