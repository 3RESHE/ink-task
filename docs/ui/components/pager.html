Props

Name Type Required Notes total Number Yes Total number of items to paginate. start Number No Start offset for pagination. range Number No Number of items per page. radius Number No Determines the number of page links to show around the current page. next Boolean No Flag to show the "next" button. prev Boolean No Flag to show the "previous" button. rewind Boolean No Flag to show the "rewind" button. forward Boolean No Flag to show the "forward" button. link String No Custom color for the pagination link. control Object No Custom styles for pagination control elements. border String No Border color for pagination controls. background String No Background color for pagination controls. square Number No Defines the size of pagination controls if set. size String No Default size of the icon for pagination controls. xs, sm, md, lg, xl, xl2, xl3, xl4, xl5 String No Various sizes for icon (pass to icon component). color String No Default color for pagination controls. white, black, info, warning, success, error, muted, primary, secondary String No Color options for pagination components. page (select) Function Yes Handler function to select a page.

Pagination Example

Pagination allows you to control the navigation through multiple pages with customizable styles and controls. Below is an example of a fully featured pagination component.
<element-pager total={1000} range={50} start={0} show={5} next prev rewind forward white bold underline bg-info border-theme="bd-2" square={40} select={console.log} />

Custom Range and Page Selection

Customize the range of items per page and the page selection. Here is an example where the pagination starts from the 90th item, with 100 items per page and a display of 3 pages at a time.
<element-pager total={500} range={100} start={90} show={3} next prev rewind forward white bold underline bg-warning border-theme="bd-2" square={40} select={console.log} />

Custom Controls and Styles

Pagination can have various styles for buttons like square and customizable borders. This example shows a pagination with square controls and different background colors for pagination elements.
<element-pager total={500} range={50} start={0} show={5} next prev rewind forward white bold underline bg-success border-theme="bd-3" square={40} select={console.log} />

Pagination with No Forward or Rewind

You can control whether to show the forward or rewind buttons using the corresponding props. This example disables forward and rewind buttons.
<element-pager total={500} range={50} start={0} show={5} next prev white bold underline bg-danger border-theme="bd-4" square={40} select={console.log} />
