Notify

import Notify from '@stackpress/ink-ui/element/notify';

Props

Name Type Required Notes top Boolean No Positions the notification at the top (default is bottom). left Boolean No Aligns the notification to the left (default is right). center Boolean No Centers the notification horizontally. fade Boolean No Enables fade transition effect. smooth Number No Controls transition smoothness (default: 10ms). type String Yes Defines the notification type: "info", "warning", "error", "success", "primary", "secondary", "muted". message String Yes Sets the notification message. timeout Number No Duration before the notification disappears (default: 5000ms).

Notification Variants

Notifications can have different variants: info, warning, success, error, primary, secondary, muted.
Show Info Show Warning Show Success Show Error
<element-button onclick="document.querySelector('element-notify').notify('info', 'Information alert!')"> Show Info </element-button> <element-notify/>

Customizing Notification Position

Set notifications to appear at different positions using top, left, or center.
Show Top Show Right Show Left Show Center
<div class="grid gap-10"> <element-button onclick="document.querySelector('element-notify[top]').notify('info', 'Notification at the top!')"> Show Top </element-button> <element-button onclick="document.querySelector('element-notify[right]').notify('info', 'Notification at the top!')"> Show Right </element-button> <element-button onclick="document.querySelector('element-notify[left]').notify('info', 'Notification on the left!')"> Show Left </element-button> <element-button onclick="document.querySelector('element-notify[center]').notify('info', 'Centered notification!')"> Show Center </element-button> </div> <element-notify top/> <element-notify right/> <element-notify left/> <element-notify center/>

Custom Timeout

Control how long notifications stay visible using a timeout value.
Show 8s Notification Show 3s Notification
<element-button onclick="document.querySelector('element-notify').notify('info', 'This disappears in 8 seconds!', 8000)"> Show 8s Notification </element-button> <element-notify/>

Dismissible Notifications

Notifications can be manually dismissed using a close button.
Show Dismissible
<element-button onclick="document.querySelector('element-notify').notify('info', 'Click the close icon to dismiss!')"> Show Dismissible </element-button> <element-notify/>

Notifications can include icons using interface-icon.
